最後まで読んでから開始してください。所要時間は **1〜2 時間** を見込んでください。

## 1. はじめに（準備するもの）

### 用意するもの
- パソコン（Windows / Mac / Linux どれでも可）
- インターネット接続
- **Chrome** または **Edge** ブラウザを推奨
  - Firefox / Safari でも使えますが、一部機能に制約があります

### テスターとしてやっていただきたいこと
- アプリを使って、動くところ・動かないところ・使いづらいところを報告
- 感想（わかりやすい・むずかしい・こうしたい等）も大歓迎

### 用意されているファイル
テスターに渡されているフォルダに以下が入っています:
- **TEST-GUIDE.md** （このファイル）
- **CHECKLIST.md** （確認項目の一覧表）
- **samples/** （テスト用のサンプルファイル 7 種類）

## 2. アプリを開く

1. 配布された URL をブラウザで開きます（Chrome か Edge 推奨）
2. 数秒待つと、画面に図が表示されます
3. 初期状態で以下のような TEM 図が表示されているはずです:
   - **出発点** → **必須通過点** → **分岐点** → **等至点** / **両極化等至点**
   - **SD** / **SG** の五角形
   - **時期1 / 時期2** の帯
   - 画面下に **非可逆的時間** の矢印

これが出ていれば、アプリは正常に起動しています。

## 3. 画面の各部分の名前

説明を理解しやすくするため、画面のどこに何があるかを覚えておいてください。

```
 ┌────────────────────────────────────────────────────┐
 │  リボン（ファイル / 編集 / 挿入 / 表示 / ヘルプ）    │ ← ①
 ├────────────────────────────────────────────────────┤
 │  リボンの中身（ボタンが並んでいる）                  │ ← ②
 ├──────┬──────────────────────────────┬──────────────┤
 │      │                              │              │
 │ デー │                              │ プロパティ   │
 │ タシ │     キャンバス（作図画面）    │  パネル      │
 │ ート │                              │              │
 │ パネ │   ここに TEM 図が表示される   │  選択した    │
 │ ル   │                              │  要素の設定  │
 │      │                              │              │
 │ ④   │              ③               │      ⑤       │
 ├──────┴──────────────────────────────┴──────────────┤
 │  シートタブ（ Sheet1 / Sheet2 … ）                  │ ← ⑥
 ├────────────────────────────────────────────────────┤
 │  ステータスバー（拡大率 / 選択数 等）               │ ← ⑦
 └────────────────────────────────────────────────────┘
```

### ① リボンのタブ
画面上部の **「ファイル / 編集 / 挿入 / 表示 / ヘルプ」** という文字部分。クリックすると下の内容（②）が切り替わります。

### ② リボンの中身
タブで選んだ機能ボタンが並びます:
- **ファイル**: 新規作成 / 開く / 保存 / CSV インポート / 出力 / 論文レポート / 設定
- **編集**: コピー / 貼付 / 削除 / 元に戻す / 全選択 / 整列 / サイズ揃え / シートリサイズ / 一括移動
- **挿入**: 各種 Box（通常 / 分岐点 / 等至点 等）/ 矢印 / SD/SG / 時期ラベル
- **表示**: グリッド / スナップ / 用紙枠 / ルーラー / 凡例 / 時間矢印 / fit ボタン群
- **ヘルプ**: バージョン情報

### ③ キャンバス
中央の広い領域。ここに TEM 図が描かれます。ここで Box をドラッグしたり、クリックして選択したりします。

右下に小さなボタンがあります:
- **＋ −**: 拡大・縮小
- **⛶**: 全体を画面に収める
- **↔**: 横幅に合わせる
- **↕**: 縦幅に合わせる
- **🔓/🔒**: 編集のロック切替

### ④ データシートパネル（左側）
データが表のように並んでいます。Box を追加すると行が増えます。
- 左上の「×」で折りたため、折りたたんだ状態の細い帯をクリックで再表示

### ⑤ プロパティパネル（右側）
Box や矢印をクリックで選ぶと、その設定がここに出ます。
- **タブ構成**（Box を選んだとき）: 「基本」「装飾・調整」「論文」
- 「基本」: ラベル / 種類 / 位置 / サイズ 等
- 「装飾・調整」: タイプラベル / サブラベル / 自動調整 等
- 「論文」: 論文レポート出力用の説明文
- 凡例をクリックすると凡例の設定、何も選んでないと説明文が出ます

### ⑥ シートタブ
画面下の「Sheet1」などの帯。1 つのファイルに複数のシート（TEM 図）を持てます。タブ右クリックで追加・削除等。

### ⑦ ステータスバー
最下部。拡大率や選択中の要素数などの情報。

## 4. まずは触ってみよう（5 分）

### 4-1 Box をクリックする
- 画面中央の「出発点」という Box をクリック
- Box に青い枠が出る → 選択された状態
- 右側の**プロパティパネル**に Box の情報が出る

### 4-2 Box をドラッグで動かす
- 選択した Box を別の場所にドラッグ
- 近くの Box と端が揃うと、**ピンクの点線**が一瞬出る（スマートガイド）

### 4-3 Box のラベルを変える
- Box を**ダブルクリック**すると、中身が文字入力になる
- 好きな文字を打って **Enter** で確定
- 間違えたら **Esc** でキャンセル

### 4-4 Box のサイズを変える
- Box を選択すると、角と辺の中点に**青い小さな四角（ハンドル）** が出る
- ハンドルをドラッグして大きさを変える

### 4-5 保存してみる
- 画面上の **「ファイル」** タブをクリック
- **「保存」** ボタンをクリック
- Chrome / Edge: ファイル保存ダイアログが出る → 任意の場所に保存
- Firefox / Safari: 勝手にダウンロードされる

これで基本操作を一通り体験できました。

## 5. チェックリストの進め方

同じフォルダの `CHECKLIST.md` を開いてください。
21 個のカテゴリに分かれた確認項目があります。

### チェックのやり方
各項目には `[ ]` という空欄があります。動作を確認できたら `[x]` のように書き換えてください。

例:
```
変更前: - [ ] 初期表示でサンプル図が出ている
変更後: - [x] 初期表示でサンプル図が出ている
```

うまく動かなかった項目はそのまま `[ ]` で残し、**「あとで Google Form で報告」** しやすいよう別紙にメモしておくと便利です。

### どこまでやればいい？
できれば全部、でも大変なら「Box」「Line」「ファイル保存」あたりの基本動作優先でお願いします。

## 6. 機能ごとのくわしい動作確認

以下は**特に確認してほしい機能**です。順番通りにやってください。

### 6-1 Box を追加する
1. 画面上の **「挿入」** タブ
2. **「通常」** ボタンをクリック
3. 画面に新しい Box が追加される
4. **「等至点」** や **「分岐点」** なども試す
5. それぞれ違う枠の種類（二重線・太線・点線 等）になっていることを確認

### 6-2 Box の文字装飾
1. Box を選択
2. プロパティパネルの **「基本」** タブで「ラベル」欄の入力ボックスを探す
3. その上に **B / I / U / S / Aa / 🎨 / Fn** のボタンが並んでいる
4. ラベル欄で文字を選択 → **B** ボタン → その部分が太字に
5. **🎨** ボタン → 色を入力（例: `#cc0000`）→ 赤に
6. キャンバスの Box を見て、太字や色が反映されているか確認

### 6-3 矢印をつなぐ
1. 2 つの Box を用意
2. **「挿入」** タブの「実線矢印」または「点線矢印」を使う場合:
   - 両方の Box を選択（1 つ目 → Shift キー押しながら 2 つ目）
   - 「実線矢印」ボタンをクリック → 矢印が追加される
3. もうひとつの方法:
   - Box の端にカーソルを合わせるとハンドル（小さな点）が出る
   - ハンドルから他の Box へドラッグして接続

### 6-4 SD / SG の追加
1. 1 つの Box を選択
2. **「挿入」** タブで **「SD」** または **「SG」** ボタン
3. Box の近くに五角形が追加される
4. 選択したまま位置を調整できる

### 6-5 CSV を読み込む
1. **「ファイル」** タブ → **「CSV インポート…」**
2. **「ファイルを選択」** をクリック
3. `samples/labels-only.csv` を選ぶ
4. 列の割り当てを確認
5. **「インポート」** ボタン → Box が追加される

**次に複雑な CSV:**
1. 別シートを作成（画面下のシートタブを右クリック → 「シートを追加」）
2. CSV インポート → `samples/20-boxes-complex.csv` を選ぶ
3. 20 個の Box が追加される
4. アプリが重くならないか、表示が崩れないか確認

### 6-6 サンプル .tem ファイルを開く
1. **「ファイル」** タブ → **「開く (Ctrl+O)」**
2. `samples/sample-simple.tem` を選ぶ
3. 既存の図が上書きされず、新しい TEM 図が表示される
4. 同様に `sample-vertical.tem` も試す（縦型レイアウトに切り替わる）

### 6-7 レイアウトの切り替え
1. 画面上の **「横型 / 縦型」** のような切替ボタンを探す（リボン右上あたり）
2. クリックで切替
3. 図が自動で縦向き/横向きに組み替えられる
4. 時間矢印の方向も変わる

### 6-8 出力してみる（エクスポート）
1. **「ファイル」** タブ → **「出力…」**
2. 出力プレビュー画面が開く
3. 左側: さまざまな調整項目
4. 右側: プレビュー
5. 下部のタブで **PNG / SVG / PDF / PPTX** を切替
6. 各タブで **「（形式）で保存」** ボタンを押す
7. 画像・PDF・プレゼンファイルがダウンロードされる
8. ダウンロードしたファイルをダブルクリックで開いて中身を確認

### 6-9 論文レポート
1. **「ファイル」** タブ → **「論文レポート…」**
2. タブを一つずつ開いて入力:
   - **概要**: タイトル / 著者
   - **記号体系**: 「荒川・安田・サトウ 2012」を選ぶ
   - **協力者**: 人数 / 記述
   - **インタビュー**: 方法 / 期間
   - **表記方針**: 「横軸の長さ」にチェック + 説明
3. **出力** タブ → **「レポートを生成」**
4. `.docx` ファイルがダウンロードされる
5. Word で開いて中身を確認

### 6-10 シート全体のリサイズ
1. **「編集」** タブ → **「シート > リサイズ…」**
2. 「パーセント指定」→ 150 と入力 → プレビュー表示
3. **「リサイズを適用」** → シート全体が 1.5 倍に
4. **Ctrl+Z** で元に戻る

### 6-11 一括移動
1. **「編集」** タブ → **「シート > 一括移動…」**
2. Time_Level 方向に **2**、Item_Level 方向に **1**
3. **「一括移動」** → すべての Box が右上に移動
4. 時期区分・時間矢印・凡例は動かないことを確認

### 6-12 設定
1. **「ファイル」** タブ → **「設定」**
2. タブ切替で各種設定を確認:
   - 全体 / スナップ / タイプラベル / 時間矢印 / 凡例 / 時期区分 / プロジェクト
3. ダイアログの **タイトル部分をドラッグ** してウィンドウを移動できる
4. 用紙サイズを A3 に変えて、表示タブの「用紙枠」ON → A3 の枠が表示される

### 6-13 凡例の編集
1. キャンバス右下の **凡例** をクリック
2. プロパティパネルに凡例設定が表示される
3. タイトル位置を「左側」に変えてみる
4. 列数を「2」に変えてみる
5. **凡例をダブルクリック** でも設定ダイアログが開く

## 7. 困ったときに

### アプリが重くなった / 固まった
- F5 キーで再読み込み
- Ctrl+F5 で強制再読み込み（キャッシュを使わない）

### 保存したファイルが見つからない
- Chrome / Edge: ダウンロードフォルダ、または保存時に指定した場所
- ファイル名: アプリで指定しなければ「TEMer.tem」など既定名

### 自動バックアップ
- アプリは 30 秒ごとにブラウザ内に自動バックアップしています
- 間違って閉じても、次に開いた時に「復元しますか？」と聞かれるはず

### 誤って Box を消してしまった
- **Ctrl+Z** で元に戻る
- 連打で何回でも戻せます

## 8. フィードバックの送り方

配布された **Google Form の URL** にアクセスして回答してください。

### 報告してほしいこと
- 動かなかった機能（CHECKLIST.md のどの項目か）
- 再現手順（「〇〇ボタンを押したら…」）
- ブラウザ名（Chrome / Edge / Firefox / Safari）
- OS（Windows / Mac / ...）
- できればスクリーンショット
- 感想・要望（どんな小さなことでも）

### 再現手順の書き方例
```
【問題】CSV を読み込んでもエラーが出る

【手順】
1. ファイル > CSV インポート
2. 20-boxes-complex.csv を選択
3. 列割り当て画面で「種別」列が自動認識されない

【期待】「type」→ 種別に自動で割り当てられる
【実際】すべての列が「無視」になっている

【環境】Chrome, Windows 11
```

細かい再現手順ほど、開発者は修正しやすくなります。
