## 1. Box 作成・編集
### 1-1 追加
- [ ] リボン「挿入」タブで Box 6 種（Item/BFP/EFP/P-EFP/OPP/annotation）を追加できる
- [ ] 2nd-EFP / P-2nd-EFP も追加できる
- [ ] 既存 Box が選択されているとき、その位置を基準に追加される

### 1-2 編集
- [ ] Box を**ダブルクリック**で textarea 表示、インライン編集できる
- [ ] Enter で確定、Esc でキャンセル、blur でも確定
- [ ] Box 選択時に 8 点**リサイズハンドル**が出る
- [ ] リサイズで Box のサイズが更新される
- [ ] PropertyPanel の「基本」タブで ID / ラベル / 種別 / 位置 / サイズ を編集できる
- [ ] 種別を変更すると ID が自動更新される（単一選択時）

### 1-3 自動調整
- [ ] 「Box を文字に合わせる」ボタンで Box が最小サイズに（**左辺中点固定**で動かない）
- [ ] 「文字を Box に合わせる」ボタンで文字サイズが Box に fit
- [ ] `autoFitText` ON で自動的に fit（縮小も拡大も）、autoFitBoxMode は停止する
- [ ] `autoFitBoxMode: width-fixed` で高さが自動拡張（左辺中点固定）
- [ ] `autoFitBoxMode: height-fixed` で横幅が自動拡張
- [ ] 全体設定「Box 自動調整（既定）」で既定値が切替わる

### 1-4 装飾・調整タブ
- [ ] 幅揃え / 高さ揃え / サイズ揃え / 文字サイズ揃え（複数選択時）
- [ ] タイプラベル / サブラベルの編集（サイズ・フォント・B/I/U、ASCII 縦向き）
- [ ] ID バッジの位置・サイズ

### 1-5 論文タブ
- [ ] 説明文（description）編集
- [ ] 「説明不要」チェックボックス

## 既知の制限
- PPTX の altText 識別は使わずマーカー色 `#2E2D2D` 方式で XML ポストプロセス
- 縦書き Box の autoFitText は二分探索が遅めになる可能性
- 大規模図（> 100 Box）での React Flow パフォーマンスは未検証
