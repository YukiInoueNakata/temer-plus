# TEMer Plus ユーザーマニュアル

TEMer Plus は TEM（複線径路等至性モデル: Trajectory Equifinality Model）の作図を支援する Web アプリケーションです。ブラウザ上で TEM 図の作成・編集・出力ができます。

---

## 1. 画面構成

アプリを開くと、以下の5つの領域が表示されます。

```
+----------------------------------------------------------+
| リボン（タブ切替式メニューバー）                            |
+--------+-----------------------------------+-------------+
|        |                                   |             |
| データ |       キャンバス（作図領域）         | プロパティ   |
| シート |                                   | パネル      |
|        |                                   |             |
+--------+-----------------------------------+-------------+
| シートタブ                                                |
+----------------------------------------------------------+
| ステータスバー                                             |
+----------------------------------------------------------+
```

### 1.1 リボン

画面上部のメニューバー。5つのタブ（ファイル / 編集 / 挿入 / 表示 / ヘルプ）で機能が分類されています。右端にはレイアウト切替（横型/縦型）、言語切替（日本語/English）、保存ボタンがあります。

### 1.2 データシート

画面左側の表形式パネル。Box / Line / SD/SG の一覧をスプレッドシート風に表示・編集できます。「表示」タブの「データ」ボタンで表示/非表示を切り替えます。

### 1.3 キャンバス（作図領域）

画面中央の白いエリア。TEM 図を視覚的に操作します。Box のドラッグ移動、ダブルクリックでのラベル編集、矢印の接続など、作図の主要操作はここで行います。

キャンバス右下にはコントロールボタン群（ズームイン/アウト、フィット、編集ロック）があります。

### 1.4 プロパティパネル

画面右側。選択中の図形の詳細設定を行います。「表示」タブの「プロパティ」ボタンで表示/非表示を切り替えます。

### 1.5 シートタブ

画面下部。複数シートの切り替えと管理を行います。「+」ボタンで新規シート追加。タブの右クリックで操作メニューが出ます。

### 1.6 ステータスバー

最下部の情報表示バー。ズーム率、グリッド ON/OFF、スナップ ON/OFF、レイアウト方向、選択中の要素数、保存状態を表示します。

## 2. リボン（メニュー）

### 2.1 ファイル タブ

ファイルの入出力と環境設定。

| ボタン | 機能 |
|--------|------|
| 新規 (Ctrl+N) | 空のシートで新規作成（未保存の変更がある場合は確認ダイアログ） |
| 開く (Ctrl+O) | `.tem` ファイルを開く（現在のデータは破棄） |
| 別シートとして開く | `.tem` ファイルを開き、そのシートを現在のドキュメントに追加（既存シートはそのまま） |
| 保存 (Ctrl+S) | 現在のファイルに上書き保存。Chrome/Edge ではネイティブ上書き。Safari/Firefox はダウンロード形式 |
| 名前を付けて保存 | 新しいファイル名で保存 |
| CSV インポート... | CSV ファイルから Box を一括追加するダイアログを開く |
| 出力... | PNG / SVG / PDF / PPTX の出力プレビューダイアログを開く |
| 論文レポート(調整中) | 現在一時停止中。クリックすると案内メッセージが表示 |
| 設定 | 設定ダイアログを開く |

### 2.2 編集 タブ

図形の操作・配置・サイズ調整。

#### モードグループ

| ボタン | 機能 |
|--------|------|
| 移動 | ドラッグで画面をパン（スクロール）。図形のドラッグ移動・リサイズはできないが、ダブルクリックでの文字編集は可能 |
| 選択 | 図形のクリック・ドラッグ・リサイズが自由にできる標準的な編集モード |
| 範囲選択 | ドラッグで矩形を描き、範囲内の図形をまとめて選択 |

#### クリップボードグループ

| ボタン | 機能 |
|--------|------|
| コピー | 選択中の Box / Line / SD/SG をクリップボードにコピー（Ctrl+C） |
| 貼付 | クリップボードの内容を貼り付け（Ctrl+V）。少しずらした位置に配置 |
| 削除 | 選択中の要素を削除（Delete / Backspace） |

#### 履歴グループ

| ボタン | 機能 |
|--------|------|
| 元に戻す (Ctrl+Z) | 直前の操作を取り消す |
| 進む (Ctrl+Y) | 取り消した操作をやり直す |

#### 編集グループ

| ボタン | 機能 |
|--------|------|
| 順序入替(直接) | 2つの Box を選択した状態で、位置と直接接続する Line の From/To を入れ替える（イベント順序の修正用） |
| 順序入替(全リンク) | 2つの Box を選択した状態で、位置に加えて全ての Line / SD/SG / Comment の参照先も交換する |
| 以降シフト | 1つの Box を選択した状態で、指定レベル数だけその Box より後ろにある全 Box を一括移動する |
| 複製 | 選択中の要素をコピー＆ペースト（Ctrl+D） |
| 全選択 | アクティブシートの全 Box を選択（Ctrl+A） |
| 文字に合わせる | 選択 Box のサイズ（幅・高さとも）をラベル文字数に合わせて最小化 |
| 幅を文字に | 高さは維持し、幅のみをラベルに合わせて調整 |
| 高さを文字に | 幅は維持し、高さのみをラベルに合わせて調整 |

#### 整列グループ

2つ以上の Box を選択した状態で使用。

| ボタン | 機能 |
|--------|------|
| 左揃え | 選択 Box の左端を最も左の Box に揃える |
| 水平中央 | 選択 Box の中心 X を平均位置に揃える |
| 右揃え | 選択 Box の右端を最も右の Box に揃える |
| 上揃え | 選択 Box の上端を最も上の Box に揃える |
| 垂直中央 | 選択 Box の中心 Y を平均位置に揃える |
| 下揃え | 選択 Box の下端を最も下の Box に揃える |
| 等間隔(横) | 選択 Box を横方向に等間隔で配置 |

#### サイズ統一グループ

2つ以上の Box を選択した状態で使用。基準は最初に選択した Box。

| ボタン | 機能 |
|--------|------|
| 幅を揃える | 選択 Box の幅を先頭の Box に揃える |
| 高さを揃える | 選択 Box の高さを先頭の Box に揃える |
| サイズを揃える | 幅と高さの両方を揃える |
| 文字サイズ揃え | 選択 Box のフォントサイズを先頭の Box に揃える |

#### 順序グループ

重なり順の変更。

| ボタン | 機能 |
|--------|------|
| 最前面 | 選択中の要素を全ての前面に |
| 前面 | 1つ前に |
| 背面 | 1つ後ろに |
| 最背面 | 全ての背面に |

#### シートグループ

| ボタン | 機能 |
|--------|------|
| リサイズ... | シート全体のリサイズダイアログを開く |
| 一括移動... | 時期区分・時間矢印・凡例を除く全要素を Time/Item 方向に一括移動するダイアログ |

### 2.3 挿入 タブ

新しい図形の追加。

#### Box グループ

| ボタン | 機能 |
|--------|------|

| 潜在経験 | 注釈 Box（細い点線枠）を追加 |

#### 2選択間に グループ

| ボタン | 機能 |
|--------|------|
| 間に挿入... | 2つの Box を選択した状態で、間に新しい Box を挿入するダイアログを開く |

#### Line / 矢印 グループ

2つの Box を選択した状態で使用。選択順に矢印を引きます。

| ボタン | 機能 |

| 点線矢印 | XLine（点線の矢印）を追加 |

| 順次接続(点線) | 同上、点線矢印で連結 |

#### SD/SG グループ

Box または Line を1つ選択した状態で使用。選択した要素に紐づきます。

| ボタン | 機能 |
|--------|------|
| SD追加 | SD（Social Direction、社会的方向づけ）を追加。五角形、下向きの力 |
| SG追加 | SG（Social Guide、社会的ガイド）を追加。五角形、上向きの力 |

#### その他グループ

| ボタン | 機能 |
|--------|------|
| 時期ラベル... | 時期ラベルの追加・編集ダイアログ（PeriodLabelsDialog）を開く |

| タイプラベル | 全ての種別バッジ（OPP, BFP, EFP 等の表記）の一括表示/非表示切替 |

#### 図要素グループ

| ボタン | 機能 |

## 3. 設定ダイアログ

「ファイル」タブの「設定」ボタン、または各図要素のダブルクリックで開きます。7つのタブがあります。ドラッグでダイアログの位置を移動できます。

### 3.1 全体 タブ

アプリケーション全体の基本設定。

| 項目 | 説明 |
|------|------|
| レイアウト | 図の向き。「横型」は左→右に時間が流れる。「縦型」は上→下に流れる |
| 言語 | UI の表示言語（日本語 / English） |
| UI フォントサイズ (px) | アプリ全体の基本文字サイズ（10〜40px、既定 13px） |
| リボン文字サイズ (px) | リボンのボタンラベルの文字サイズ（8〜24px、既定 12px） |
| Level 調整刻み | プロパティパネルで Item/Time レベルを矢印キーやスピナーで調整する際の刻み幅（既定 0.5） |

#### 用紙枠

| 項目 | 説明 |
|------|------|
| 用紙サイズ | 用紙枠の基本サイズ。A4 / A3 / 16:9 / 4:3 / カスタム |
| カスタム短辺/長辺 | 「カスタム」選択時に短辺・長辺をピクセル単位で指定 |
| 枚数（長辺方向） | 複数枚並べる場合の枚数（1〜50） |
| 枠外を薄グレーで表示 | 用紙枠の外をグレーでマスクするかどうか |
| 枠線の色 | 用紙枠の線の色 |

#### Box 自動調整（既定）

| 項目 | 説明 |
|------|------|
| 自動拡張モード | Box 個別設定がない場合のデフォルト動作。「横幅固定で高さ自動」= ラベルが収まらない時に高さを伸ばす。「高さ固定で横幅自動」= 幅を伸ばす |
| 文字サイズ自動調整 | Box に収まる最大文字サイズを自動設定。自動拡張モードとは同時有効不可 |

### 3.2 スナップ タブ

グリッドとスナップの詳細設定。

| 項目 | 説明 |
|------|------|
| グリッドを表示 | キャンバスの背景ドットを表示するかどうか |
| スナップを有効化 | Box ドラッグ時にグリッドに吸着させるかどうか |
| グリッド (px) | グリッドの間隔（ピクセル単位、既定 10px）。25〜50 にするとスナップの効きを体感しやすい |
| 他 Box と整列時にガイド線を表示 | Box ドラッグ中に他の Box と端が揃った時にピンクの点線を表示 |
| 距離スナップ (px) | 一定距離に近づいた時にスナップする閾値 |

### 3.3 タイプラベル タブ

各種別バッジの表示/非表示を個別に設定。

| 項目 | 説明 |
|------|------|
| 一括: すべて表示 / すべて非表示 | 全種別を一括で切替 |
| BFP（分岐点） | BFP のタイプラベルを表示するか |
| EFP（等至点） | EFP のタイプラベルを表示するか |
| P-EFP（両極化等至点） | P-EFP のタイプラベルを表示するか |
| OPP（必須通過点） | OPP のタイプラベルを表示するか |
| 2nd-EFP（第二等至点） | 2nd-EFP のタイプラベルを表示するか |
| P-2nd-EFP（両極化第二等至点） | P-2nd-EFP のタイプラベルを表示するか |
| SD（社会的方向づけ） | SD のタイプラベルを表示するか |
| SG（社会的ガイド） | SG のタイプラベルを表示するか |

### 3.4 非可逆的時間 タブ

非可逆的時間の矢印（図の下部/左側に表示される長い矢印）の設定。キャンバス上の矢印ラベルをダブルクリックしても開きます。

| 項目 | 説明 |
|------|------|
| エクスポート時に自動挿入 | PNG / PDF / PPTX 出力時に矢印を含めるか |
| 編集中も常時表示 | キャンバス上に常に矢印を表示するか |
| 開始レベル拡張 | 矢印の開始位置を最小 Time_Level からさらに延長する値（既定 -0.5） |
| 終了レベル拡張 | 矢印の終了位置を最大 Time_Level からさらに延長する値（既定 +0.5） |
| Item軸の基準 | 矢印の Item 方向の位置基準。「最大Item_Level（上側）」= 図の上、「最小Item_Level（下側）」= 図の下 |
| 基準からのオフセット | 基準からの距離（Level 単位）。負の値で基準から離れる |
| ラベル | 矢印に表示するテキスト（既定: 「非可逆的時間」） |
| ラベル位置（横型レイアウト） | 矢印線に対するラベルの位置（上部 / 下部） |
| ラベル位置（縦型レイアウト） | 矢印線に対するラベルの位置（左側 / 右側） |
| ラベルオフセット | ラベルと矢印線との間隔（ピクセル） |
| ラベル揃え（横型） | 矢印方向のラベル位置（中央 / 端） |
| ラベル揃え（縦型） | 矢印方向のラベル位置（中央 / 開始端） |
| ラベル フォント | ラベルのフォント指定 |
| ラベル 装飾 | ラベルの太字(B) / 斜体(I) / 下線(U) |
| 線の太さ | 矢印線の太さ（ピクセル単位、既定 2.5） |
| フォントサイズ | ラベルの文字サイズ（既定 14px） |

### 3.5 凡例 タブ

凡例パネルの詳細設定。キャンバス上の凡例をダブルクリックしても開きます。

#### 表示対象

| 項目 | 説明 |
|------|------|
| 自動生成 | シート内で使用されている記号を自動抽出して凡例を生成 |
| 編集中も表示 | キャンバス上に常に凡例を表示するか |
| エクスポートに含める | 出力時に凡例を含めるか |
| Box 種別を含む | Box の種類（通常, BFP, EFP 等）を凡例に含める |
| Line 種別を含む | 矢印の種類（実線, 点線）を凡例に含める |
| SD/SG を含む | SD/SG を凡例に含める |
| 非可逆的時間を含む | 非可逆的時間の矢印を凡例に含める |

#### タイトル

| 項目 | 説明 |
|------|------|
| タイトルを表示 | 凡例の上部にタイトルバーを表示するか |
| タイトル文字列 | タイトルのテキスト（既定: 「凡例」） |
| 配置位置 | タイトルの位置（上部 / 左側） |
| 書き方向 | タイトルの書字方向（横書き / 縦書き） |
| 揃え | タイトルの水平揃え（左 / 中央 / 右） |
| 上下揃え（左側配置時） | 左側配置時の垂直揃え（上 / 中央 / 下） |
| タイトル フォント | タイトル用フォント（「本文と同じ」可） |
| タイトル サイズ | タイトルの文字サイズ（未指定時は本文の 1.15 倍） |
| タイトル 装飾 | 太字(B) / 斜体(I) / 下線(U) |

#### レイアウト

| 項目 | 説明 |
|------|------|
| 横型レイアウト時の列数 | 横型レイアウトでの凡例項目の列数（既定 1） |
| 縦型レイアウト時の列数 | 縦型レイアウトでの列数（既定 1） |
| サンプル図形 幅/高さ | 凡例内の見本図形のサイズ（ピクセル） |
| 本文 フォント | 凡例テキストのフォント |
| 本文 フォントサイズ | 凡例テキストの文字サイズ（既定 11px） |
| 最小幅 | 凡例パネルの最小幅（既定 200px） |
| 2行目（説明文）を既定で表示 | 各項目に説明文の行を表示するか |

#### サイズ（固定）

| 項目 | 説明 |
|------|------|
| 幅 (px) | 凡例の固定幅。未入力の場合は内容に合わせて自動 |
| 高さ (px) | 凡例の固定高さ。未入力の場合は自動 |

#### 背景・枠線

| 項目 | 説明 |
|------|------|
| 背景 | 「白背景」または「透明」 |
| 枠線の太さ | 枠線の太さ（0 で枠線なし） |
| 枠線の色 | 枠線の色 |

#### タイトル区切り線

| 項目 | 説明 |
|------|------|
| 表示 | タイトルと項目の間に区切り線を引くか |
| 色 | 区切り線の色 |

#### 項目別の表記

シート内に存在する各凡例項目に対して個別に以下を設定可能:

## 6. ダイアログ（ポップアップウィンドウ）

### 6.1 間に挿入ダイアログ

2つの Box を選択した状態で「挿入」タブ → 「間に挿入...」で開きます。

| 項目 | 説明 |
|------|------|
| 開始 Box (A) / 終了 Box (B) | 選択した2つの Box の情報（読み取り専用） |

| 項目 | 説明 |

| 用紙内余白比率 | 用紙端からの余白（0〜0.3、既定 0.05 = 5%） |
| 文字サイズも連動 | 拡縮時に文字サイズも比例して変更するか |

## 10. データの保存と復元

### 10.1 ファイル保存

TEMer Plus のデータは `.tem` ファイル（JSON 形式）として保存されます。

- **Chrome / Edge**: File System Access API により、ネイティブの「上書き保存」が可能
- **Safari / Firefox**: ダウンロード形式での保存（毎回ファイル名を指定）

### 10.2 自動バックアップ

30 秒ごとにブラウザ内の IndexedDB に自動保存されます。次回アプリを開いた際に復元の確認ダイアログが表示されます。

### 10.3 ブラウザ離脱警告

未保存の変更がある状態でブラウザを閉じようとすると、確認ダイアログが表示されます。

## 11. 凡例の操作

キャンバス上の凡例パネルは以下の操作ができます:

| 操作 | 動作 |
|------|------|
| クリック | 凡例を選択（プロパティパネルに設定が表示される） |
| ダブルクリック | 設定ダイアログの「凡例」タブを開く |
| 右下ハンドルをドラッグ | 凡例のサイズを変更 |
| 移動ハンドル（⠿）をドラッグ | 凡例の位置を移動 |

---

## 12. 注意事項

- **個人情報の取り扱い**: 入力したデータは全てブラウザ内に留まります。サーバーには送信されません
- **推奨ブラウザ**: Chrome / Edge（保存機能がネイティブ対応）
- **論文レポート出力**: 現在調整中のため一時停止しています
- **大規模図**: Box が 100 個を超える図のパフォーマンスは検証中です
