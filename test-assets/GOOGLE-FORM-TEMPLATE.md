# テスター用 Google Form 作成手順

自動では作成できないため、以下のテンプレを Google Form にコピペしてください。

## 1. フォーム作成手順

1. Google アカウントにログイン
2. https://forms.google.com/ で「空白のフォーム」を作成
3. タイトル: **TEMer Plus テスターフィードバック**
4. 説明文:
   ```
   TEMer Plus のテストにご協力いただき、ありがとうございます。
   使用感や不具合について回答をお願いします。
   ---
   公開 URL: https://<username>.github.io/<repo-name>/
   チェックリスト: test-assets/CHECKLIST.md
   サンプル: test-assets/samples/
   ```
5. 以下の質問を追加

---

## 2. 質問テンプレート（コピペ用）

### 基本情報

**Q1. お名前（または匿名 ID）** - 短文回答
- 必須: はい

**Q2. 所属（任意）** - 短文回答

**Q3. テスト日** - 日付形式

**Q4. 使用ブラウザ** - 複数選択
- Chrome
- Edge
- Firefox
- Safari
- その他（テキスト入力）

**Q5. OS** - 選択式
- Windows 10 / 11
- macOS
- Linux
- iOS
- Android
- その他

### 機能動作

**Q6. チェックリスト（test-assets/CHECKLIST.md）をすべて確認しましたか？** - 選択式
- はい、すべて確認した
- ほぼ確認した（8 割以上）
- 一部だけ（半分程度）
- 最低限（30% 以下）

**Q7. 動作しなかった機能をすべて選んでください** - チェックボックス（複数選択）
- Box の追加・編集
- Box のリサイズ / インライン編集
- 自動調整（autoFitText / autoFitBoxMode）
- ラベル部分装飾タグ（<b> 等）
- Line の追加・オフセット
- SDSG の追加・向き
- 時期区分
- 時間矢印
- 凡例
- スマートガイド
- レイアウト切替（横 ↔ 縦）
- ファイル保存・読込
- 自動バックアップ（復元）
- CSV インポート
- PNG 出力
- SVG 出力
- PDF 出力
- PPTX 出力
- 論文レポート (.docx)
- シートリサイズ
- シート一括移動
- 用紙枠
- 右下コントロール（Zoom / fit）
- 設定ダイアログ
- プロパティパネル
- その他
- すべて問題なく動作した

**Q8. 不具合の詳細（Q7 で「すべて問題なく動作した」以外を選んだ場合）** - 段落形式
- 説明:
  ```
  再現手順・期待動作・実際の動作・エラーメッセージなどをご記入ください。
  例:
  1. ○○ボタンを押した
  2. ○○ダイアログで〜を選択
  3. → エラー "xxx" が表示された
  ```

### UX / 使い勝手

**Q9. ユーザーインターフェースの総合評価** - 直線型（1〜5）
- 1: 非常に使いにくい
- 5: 非常に使いやすい

**Q10. 作図のしやすさ** - 直線型（1〜5）

**Q11. データ保存・読込の使いやすさ** - 直線型（1〜5）

**Q12. エクスポート機能の使いやすさ** - 直線型（1〜5）

**Q13. 論文レポート出力はニーズに合っていましたか？** - 選択式
- はい、そのまま使える
- 概ね使えるが修正が必要
- 大幅な改善が必要
- 使わない / 評価できない

### 要望・質問

**Q14. 現時点で「最も重要な改善点」を 1 つ** - 段落形式

**Q15. 「あったら嬉しい機能」** - 段落形式

**Q16. TEM 図作成の従来手段（比較対象）** - 複数選択
- PowerPoint
- 旧版 TEMer（Excel VBA）
- Cacoo / Lucidchart 等の Web 作図
- Microsoft Visio
- Illustrator / Photoshop
- 手描き
- 使ったことがない

**Q17. 既存ツールと比較した強み・弱み** - 段落形式

**Q18. その他ご自由に** - 段落形式

### 付加情報（任意）

**Q19. 作成した .tem ファイル・スクリーンショット（任意）** - ファイルアップロード
- 許可するファイル種類: すべて
- 最大サイズ: 100MB
- 最大ファイル数: 10

## 3. 設定の推奨

- **回答者のメールアドレスを収集**: オフ（匿名で気軽に）
- **回答の送信後、別の回答を送信するリンクを表示**: オン
- **確認メッセージ**: 「ご協力ありがとうございました。フィードバックは開発者に直接送信されました。」

---

## 4. フォーム URL を配布

公開後、短縮 URL（bit.ly 等）を作成してテスターに配布。

テスター依頼文テンプレ:
```
TEMer Plus のテスターをお願いします。

[公開 URL]
https://<username>.github.io/<repo-name>/

[テスト手順]
test-assets/TEST-GUIDE.md を参照

[チェックリスト]
test-assets/CHECKLIST.md を参照（PDF 化して渡しても可）

[フィードバック送信先]
[Google Form URL]

[サンプルファイル]
test-assets/samples/ 以下の CSV / .tem

ご不明点は <連絡先> までお願いします。
```

---

## 5. 回答の確認

- Google Form の「回答」タブでスプレッドシート連携して管理
- 不具合は GitHub Issues に転記（セキュリティ情報は伏せる）
