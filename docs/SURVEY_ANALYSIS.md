## 9. 調査から得られる戦略的示唆

### 9.1 市場ニーズの確認
- **98% が専用ツールを希望** → TEMerPlus への強い需要
- 特に **3年未満の初学者が61%** → 「初学者でも使いこなせる」UXが最重要

### 9.2 競合（PowerPoint）の弱点が明確
- 大きさ調整に時間がかかりすぎ、着手の心理的ハードルが高い（#26）
- 縮小時の画質劣化（#26）
- Word→PPT変換の労力（#16）
- 分岐点追加後の体裁崩れ（#26）

→ TEMerPlus がこれらを解決すれば、PowerPoint から明確に切り替える価値がある

### 9.3 論文執筆支援としての価値
- **本文⇔図ラベル同期**、**逐語録メンション**、**A4収納チェック** は論文作成に直結
- 他の作図ツールにはない、**TEMerPlus 独自の論文執筆補助ツール**としてポジショニング可能

### 9.4 初学者向けの工夫の重要性
- 「TEMの作られ方の自由度が高すぎて悶々とした」（#33）
- **サトウ先生は画一化に反対**だが、**「複数の作法例」を提示する**ことは許容範囲
- テンプレート集 + 作法解説 + チュートリアルの3点セットが効く

### 9.5 マーケティング材料
- 46名の回答者のうち、**追加インタビュー/ユーザーテスト協力希望者が多数**
  - メールアドレス提供者 = テストユーザー候補リスト
  - 看護系・教育系・実務家など多様な領域をカバー
  - 立命館大学（TEM発祥地）の複数名が含まれる

## 10. 結論

### 10.1 既存SPECは調査で裏付けられた
**計画済み機能の大半が調査で具体的要望として挙がっており、方向性は正しい**。

### 10.2 追加すべき「キラー機能」候補
1. A4/規定サイズ収納チェック
2. 本文⇔図ラベル自動同期
3. 逐語録メンション機能
4. スペルチェック

これらは**競合ツール（PowerPoint等）にはない、TEMerPlus独自の価値**となる。

### 10.3 優先度調整の提案
- Phase 2 に「A4収納チェック」を追加（論文投稿を想定する実務的必須機能）
- Phase 3 に「スペルチェック」「本文同期」「逐語録メンション」を追加
- Phase 4 に「曲線矢印」「縦書き対応」を検討
