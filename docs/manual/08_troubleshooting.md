# 08 困ったら / よくある質問

[← マニュアルトップに戻る](./README.md)

> このページで分かること: つまずきやすい場面と、その対処法。

## まず試すこと 3 つ

何が起きていても、この 3 つを試すと多くの問題が解決します。

1. **Ctrl + Z (元に戻す)** — 直前の操作を取り消す
2. **Esc** — 選択を解除する / ダイアログを閉じる
3. **Ctrl + K** — コマンドパレットを開く (機能を検索)

## 起動・ファイル関連

### Q. アプリが開かない / 真っ白
A. ブラウザのキャッシュをリフレッシュ。**Ctrl + Shift + R** で強制リロード。
   それでも駄目なら別のブラウザ (Chrome / Edge) で試す。

### Q. 保存したファイルが見つからない
A. ブラウザのダウンロードフォルダを確認。
   通常は「ダウンロード」フォルダ (Windows: `C:\Users\<ユーザ名>\Downloads`)。
   File System Access API 対応ブラウザ (Chrome / Edge) なら、
   保存時に場所を選べるダイアログが出ます。

### Q. 自動バックアップから復元したい
A. アプリ起動時に「自動バックアップが見つかりました。復元しますか？」と聞かれます。
   「はい」を選択すれば直前の状態に戻ります。
   このダイアログを閉じてしまった場合は、**Ctrl + K → 「自動バックアップ」**
   と入力 → 該当コマンド実行。

### Q. .tem ファイルを別の PC に持って行きたい
A. `.tem` ファイルをコピーするだけで OK。中身は JSON テキストです。
   メールに添付したり、クラウドに置いたり自由に共有できます。

### Q. .tem ファイルを開こうとするとエラーになる
A. ファイルが壊れている可能性。
   - テキストエディタで開いてみて、JSON の冒頭が `{` で始まり、`}` で終わっているか確認
   - 古いバージョンで作ったファイルは多くの場合自動で互換変換されますが、ごく古いものは非対応の可能性

## 作図中のトラブル

### Q. Box が動かせない / 選択できない
A. **編集モード** が「移動禁止」になっているかも。
   リボン **Home タブ → モード** から「移動」モードを選択。

### Q. ダブルクリックで文字編集できない
A. これも編集モードによる制約。「移動」モードか「編集」モードを選んでください。

### Q. 矢印を引こうとしても丸が出ない
A. ホバー時に Box の右端 / 左端に注目してください。少し近づいてから出ます。
   それでも駄目なら、**Ctrl+クリック** で 2 つの Box を順に選択し、
   **Insert タブ → Line** から手動追加。

### Q. SD/SG を Box に紐づけたら変な位置に出た
A. 既定の位置は Box の上 (SG) または下 (SD) ですが、
   プロパティパネルの「時間オフセット」「項目オフセット」で微調整できます。
   またはマウスでドラッグで自由に位置調整可。

### Q. SDSG が「赤い枠」で警告される
A. **帯モードで帯から はみ出している** という警告です。
   PropertyPanel の上部に赤いバナー + 修正ボタン 3 つが表示されます:
   - **帯を拡張**: 帯の高さを増やす (manual + autoExpandHeight ON)
   - **SDSG を圧縮**: SDSG を row span 内に縮める
   - **row を内側へ**: 該当 SDSG の row 番号を 1 減らす

### Q. 矢印の始点 / 終点が Box の中に潜り込む
A. プロパティパネル → **端点タブ** の「始点マージン」「終点マージン」で調整。
   既定 0 で Box 端、負の値で Box から離す、正の値で内側へ。

### Q. L字 / 曲線の矢印で「どこで折れるか」を変えたい
A. 該当 Line を選択した状態で、キャンバスにオレンジ色の丸 (L字) または
   青い丸 (曲線) のドラッグハンドルが現れます。それを掴んで動かせます。

## 表示・レイアウト

### Q. 図がはみ出して見切れる
A. **View タブ → フィット** か、**Ctrl+ホイール** で縮小。

### Q. グリッドの間隔を変えたい
A. **設定 → スナップ → グリッド間隔 (px)** で変更。

### Q. 全部の Box の文字を一括で大きくしたい
A. **Ctrl + A** で全選択 → プロパティパネル → **ラベルタブ → フォントサイズ** を変更。

### Q. レイアウトを横→縦に変えたい
A. **設定 → 全体 → 表示 → レイアウト** で切替。
   既存の Box 位置は保持されますが、SDSG の配置や時間矢印の向きは
   自動で再計算されます。

## 出力関連

### Q. PNG が小さい / ぼやけて見える
A. PNG は内部で 2 倍解像度で書き出されます。さらに大きくしたい場合は、
   「微調整」セクションで **全体倍率** を上げるか、用紙サイズを A3 等に。

### Q. PowerPoint で開いたら何かが崩れている
A. 二重線の Box (EFP 等) は内部で 2 つの矩形を重ねて表現しています。
   PowerPoint で動かすときはグループ化を活用してください。
   それ以外の崩れの場合は、TEMerPlus 側で別形式 (PNG / SVG) を試して
   見比べてみてください。

### Q. SVG を Illustrator で開いたら縦書きが横倒し
A. 縦書きは `writing-mode` ではなく **1 文字 1 文字を縦に積む方式** で出力しています。
   Illustrator で開いた直後は通常立った状態のはずです。
   倒れて見える場合は Illustrator のテキスト方向設定を確認。

### Q. 出力プレビューに何も表示されない
A. キャンバスに何も無い (Box が 0 個) 場合、プレビューも空白です。
   Box を 1 つでも追加してから再試行。

### Q. 印刷で図が用紙に入りきらない
A. 「フィット: 両方に収める」を選択。
   それでも入らない場合は **複数印刷 (分割)** モードに切替えて N ページに分割。

## ID バッジ

### Q. 出力に ID バッジ (`OPP1`, `EFP2` 等の小さな文字) が出ている / いない
A. 出力プレビューの **出力設定** セクションで「**ID バッジを含める**」チェックを切替。
   既定は OFF (含めない)。
   さらに Box / SDSG / Line ごとに個別 ON/OFF も可能 (チェックを入れた状態で展開)。

### Q. 編集中の ID バッジを非表示にしたい
A. リボン **View タブ → 🏷 ID** で全 ON/OFF。
   または個別に「Box-ID」「SDSG-ID」「Line-ID」ボタン。

## 検索 / コマンドパレット

### Q. Ctrl + K を押しても何も起きない
A. ブラウザの拡張機能が Ctrl+K を奪っている可能性。
   拡張を一時無効化するか、別のブラウザで試行。

### Q. コマンドパレットで日本語入力ができない
A. 検索ボックスは IME 対応です。日本語入力モードに切り替えてから入力。
   キーワードは「英語のキーワードで」検索しても多くがヒットします (`save`, `grid` 等)。

## パフォーマンス

### Q. 動作が重い / カクつく
A. Box / SDSG が 100 個を超えると重くなる場合があります。
   - 自動バックアップ間隔を伸ばす
   - グリッドを OFF にする
   - 帯モードの編集枠 (showBorder) を OFF
   - ブラウザのタブを他に開きすぎていないか確認

### Q. アプリが固まる / 反応しなくなる
A. ブラウザを **F5 (リロード)** で再読み込み。
   未保存の編集は失われますが、自動バックアップから復元できる場合があります。

## データ・互換性

### Q. 旧バージョンの .tem ファイルは開ける？
A. ほぼすべて互換です。新しいフィールド (例: SDSG の `attachedType`)
   は読込時に自動補完されます。

### Q. 旧 Excel VBA 版 (TEMerPlus 旧版) のファイルを取り込める？
A. 直接の互換はありません。CSV エクスポートで橋渡しする方法を検討中。
   暫定: **Insert → CSV インポート** で Box データだけは取り込めます。

## 他にも困ったら

- **コマンドパレット** (Ctrl+K) で「○○」と入力 → 関連機能が出る
- **設定の検索ボックス** で関連設定が見つかる
- このマニュアルの他のページを参照
- それでも解決しない場合は、現象と再現手順を控えて開発者へ連絡

## 開発者への問い合わせ

問題報告には以下があると助かります:

1. 何をしようとしていたか (操作手順)
2. 何が起きたか (期待と異なる結果)
3. ブラウザ名・バージョン (Chrome 130 等)
4. 可能なら開発者コンソール (F12 → Console) のエラーメッセージのスクリーンショット
5. .tem ファイル (再現に必要であれば)
