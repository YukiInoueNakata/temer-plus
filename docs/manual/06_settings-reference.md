# 06 設定ダイアログを使いこなす

[← マニュアルトップに戻る](./README.md)

> このページで分かること: 設定ダイアログの全タブを「辞典」として網羅。困ったときに引いてください。

## 設定を開く

3 つの方法:

| 方法 | 操作 |
|---|---|
| リボン | **File → 設定** |
| キーボード | **Ctrl + K** → 「設定」と入力 → Enter |
| 直接 | (タブを指定して開きたい場合) **Ctrl + K** → 「設定: 凡例」のように打つと該当タブが開く |

## 画面の見方

ダイアログは 2 列レイアウト。

```
┌────────────┬──────────────────────┐
│ 検索...     │                              │
├────────────┤                              │
│ 全体        │   設定の本体エリア          │
│ スナップ    │                              │
│ ▼ 図要素    │   (選択中タブの内容)         │
│   Box スタ  │                              │
│   タイプラ  │                              │
│   非可逆的  │                              │
│   時期区分  │                              │
│   凡例      │                              │
│   SD/SG    │                              │
│ プロジェクト│                              │
└────────────┴──────────────────────┘
```

### 左サイドバー
- 上部に **検索ボックス** (タブ名や設定項目をキーワード検索)
- 9 タブ: 「全体」「スナップ」+ 「図要素」配下 6 つ + 「プロジェクト」
- 検索中はマッチしないタブが薄く表示される (クリックで強制切替も可)

### 右本体エリア
選択中タブの設定項目を表示。長いタブはスクロールできる。

## 全体タブ

### 表示 (折り畳み)
| 項目 | 説明 |
|---|---|
| **レイアウト** | 横型 (時間軸 = 横) / 縦型 (時間軸 = 縦) |
| **言語** | 日本語 / English (現在は UI のみ) |
| **UI フォントサイズ** | プロパティパネル等の文字サイズ (px) |
| **リボン文字サイズ** | リボンボタンの文字サイズ (px) |

### 編集補助 (折り畳み)
| 項目 | 説明 |
|---|---|
| **Level 調整刻み** | プロパティパネルでスピナー操作するときの刻み (Level 単位) |
| **自動拡張モード (既定)** | 新規 Box の自動拡張モード既定値 |
| **文字サイズ自動調整 (既定)** | Box に収まる最大サイズへ自動調整するか |

### 用紙枠 (折り畳み、既定閉)
画面に薄く表示される用紙ガイドの設定:
- ON/OFF
- ベース用紙サイズ (A4 / A3 / 16:9 / 4:3 / カスタム)
- 縦/横の向き
- 用紙数 (横並び 1〜20 ページ)
- 用紙間オーバーラップ
- 表示色

> 用紙枠は **作図中の目安** であって、出力結果には自動で反映されません。
> 出力時の用紙サイズは [出力プレビュー](./05_export.md) で別途指定します。

---

## スナップタブ

### グリッド
- ON/OFF / グリッド間隔 (px)

### スナップ
- グリッドにスナップ ON/OFF
- 距離スナップ (他の Box との間隔に吸着)
- 整列ガイド (赤い線で揃いを表示)

> グリッド+スナップを使うと、後で整列する手間が大幅に減ります。

---

## 図要素タブ (6 つの子タブ)

「図要素」グループ配下に 6 つの子タブが並びます。

### Box スタイル

8 種類の Box の **既定スタイル** を編集します。
タブ内タブで Box 種別を切り替えながら、各々のプリセットを設定。

各種別ごとに調整できるもの:
- **本体**: 背景色 / 枠線色 / 枠線太さ / 枠線スタイル / 文字色 / 文字サイズ / 形状 (rect / ellipse)
- **タイプラベル色**: 文字 / 背景 / 枠線 / 枠線太さ
- **サブラベル色**: 文字 / 背景 / 枠線 / 枠線太さ

> ここで設定した値は、新規 Box にも既存 Box にも自動反映されます
> (個別 Box で上書きしていない限り)。

### タイプラベル

種別ラベル (OPP-1 / EFP / SD1 等) の表示を、種別ごとに ON/OFF。

- BFP / EFP / P-EFP / OPP / 2nd-EFP / P-2nd-EFP / SD / SG の 8 種類
- それぞれを個別チェックで表示・非表示
- 「**一括表示切替**」(リボン View タブ) で全種別を同時切替も可

### 非可逆的時間 (時間矢印)

図全体を貫く「時間が後戻りしない」を表す矢印の設定。

- **自動挿入**: ON にすると、出力時に時間矢印が自動で描かれる
- **常時表示**: 編集中もキャンバスに表示するか
- **位置**: minTimeLevel からの開始 / 終了オフセット
- **アイテム軸位置**: 上 / 下 のどちらに矢印を置くか
- **ラベル**: 「非可逆的時間」等のテキスト
- **線太さ・フォントサイズ・装飾 (B/I/U)**: 視覚調整
- **ラベル位置**: 矢印の上下 / 左右、矢印方向の center/start/end

### 時期区分

時間軸を区切る期間ラベル (例: 「高校 2 年生」「夏休み」) を設定。

- **常時表示** / **エクスポート時の含有**
- **基準軸**: min / max Item Level からの距離
- **フォントサイズ**
- **区切り線の表示** + **線の太さ**
- **描画スタイル**: tick (短い線+ラベル) / band (帯状)
- **ラベル位置**: 横型は上/下、縦型は左/右

期間ラベル本体の編集 (例: 「2024年4月から」を「大学入学」に変える)
は **Insert タブ → 時期ラベル...** ダイアログから。

### 凡例

図の凡例 (legend) の見た目を細かく設定。
- タイトルの表示 / 文字 / 装飾 / 位置
- 表示する項目カテゴリ (Box / Line / SDSG / 時間矢印)
- 列数 (横型 / 縦型レイアウト別)
- サンプル図形のサイズ
- 背景・枠線
- 項目別の上書き (折り畳み式) — 各項目のラベルや説明文を個別に変更

### SD/SG 配置 (帯モード設定)

SD/SG を「帯」に整列する機能の詳細設定。

#### 全体
- ON/OFF
- 一括自動配置 (新規 SD / SG をどこに置くか既定)
- 種別と帯の組合せ制限を解除するか

#### 上部 (SD) 帯 / 下部 (SG) 帯 — 各々独立に
| 項目 | 説明 |
|---|---|
| **有効** | この帯を使うか |
| **高さモード** | auto (内容から自動) / manual (heightLevel を使用) |
| **heightLevel** | 帯の高さ (Level 単位) |
| **基準** | period (時期区分) / timearrow (時間矢印) / boxes (Box 群) |
| **offsetLevel** | 基準からの距離 |
| **編集中に枠表示** | 帯の範囲を点線表示 |
| **枠線色 / 塗りスタイル** | 視覚調整 |

#### 共通
- **自動整列**: 重なる SDSG を縦積みで自動配置
- **shrinkToFitRow**: row span に合わせて SDSG 高さを圧縮
- **autoExpandHeight (manual モード時)**: row が多いとき帯を自動拡張
- **band 位置で方向点を自動反転**: SD を下帯に置いたら矢印を上向きにする等

> このタブは複雑ですが、初めは「**帯 ON**」「**自動整列 ON**」だけで十分使えます。
> 細かい調整は必要になったときに参照してください。

## プロジェクトタブ

### ドキュメント情報
- **タイトル** / **作成日** / **編集履歴件数** (読み取り専用)

### 既定値
| 項目 | 説明 |
|---|---|
| **既定 Box 幅 / 高さ** (px) | 新規 Box のサイズ。プリセットボタンあり (縦長 / 横長 / 正方形) |
| **既定フォント** | 新規 Box の文字フォント |
| **既定サイズ** | 新規 Box の文字サイズ |

> ここを変更しても、**既存** の Box には影響しません。
> 既存も変えたい場合は、Ctrl+A で全選択してプロパティパネルから一括変更。

---

## 検索の使い方

サイドバー上部の検索ボックスに任意のキーワードを入れると、
マッチしないタブが薄く表示され、関連タブが目立ちます。

例:
- 「**Box**」 → 全体 / Box スタイル / プロジェクト
- 「**帯**」 → SD/SG 配置
- 「**フォント**」 → 全体 / Box スタイル / タイプラベル / 凡例 / プロジェクト
- 「**色**」 → Box スタイル / 凡例 / 時期区分

検索文字を消すと全タブが復活します。

---

## 設定はどこに保存される？

設定は 2 種類のレイヤーがあります。

| レイヤー | 保存先 | スコープ |
|---|---|---|
| **プロジェクト固有** (例: レイアウト・凡例設定) | `.tem` ファイル | このファイルだけに適用 |
| **ユーザー固有** (例: 折り畳みの開閉状態) | ブラウザの localStorage | このブラウザだけ覚えている |

`.tem` ファイルを別 PC に持って行けばプロジェクト設定は引き継がれますが、
折り畳みの開閉状態のような UI 状態は引き継がれません (それは PC ごとの好みのため)。
