# 01 TEMerPlus を起動する

[← マニュアルトップに戻る](./README.md)

> このページで分かること: アプリの起動、画面各部の名前、最初に覚えたい操作。

## 起動する

TEMerPlus は **Web アプリ** なので、インストールは不要です。
**Google Chrome** または **Microsoft Edge** で URL を開くだけで使えます。
(Safari や Firefox でもほとんどの機能は動きますが、Chrome / Edge を推奨)

> **PC が苦手な方への補足**:
> ブラウザとは、Web ページを見るためのソフトです。
> Windows パソコンなら、デスクトップやスタートメニューに
> 青い四角に丸い線が入ったアイコンの「Microsoft Edge」、
> あるいは緑・赤・黄の輪っかが描かれた「Google Chrome」があるはずです。
> どちらかをダブルクリックで開いてから、URL 欄に管理者から共有された
> アドレスを入力してください。

起動すると、ブラウザのウィンドウいっぱいに TEMerPlus の画面が表示されます。

## 画面の見方 (5 つのエリア)

最初は情報量が多く感じるかもしれませんが、画面は次の 5 つのエリアに分かれています。

```
┌────────────────────────────────────────────────────────┐
│ ① リボン (上部の操作バー、複数のタブで分かれている)        │
├────────────────────────────────────────────────────────┤
│┌──────┬──────────────────────────┬──────┐│
││         │                                │         ││
││ ②     │ ③                          │ ④   ││
││ データ│ キャンバス (描画エリア)         │ プロパ ││
││ シート│                                │ ティ   ││
││         │                                │         ││
│└──────┴──────────────────────────┴──────┘│
├────────────────────────────────────────────────────────┤
│ ⑤ 下部 — シートタブ + ステータスバー                      │
└────────────────────────────────────────────────────────┘
```

### ① リボン (上部)

リボンには **5 つのタブ** があり、機能ごとに分かれています:

| タブ | 主な機能 |
|---|---|
| **File** | 新規作成 / 開く / 保存 / 出力 / 設定を開く |
| **Home** | 編集、選択、複製、整列、サイズ統一、シート切替など |
| **Insert** | Box / Line / SD/SG の追加 |
| **View (表示)** | グリッド・スナップ・ID バッジ・ルーラーなどの表示切替 |
| **Help** | バージョン情報など |

> リボンのボタンが見当たらないときは、別のタブに隠れているかもしれません。
> タブをクリックして切り替えてみましょう。
>
> あるいは **Ctrl+K** ([07 コマンドパレット](./07_command-palette.md))
> でキーワード検索すれば、機能が一発で見つかります。

### ② データシート (左)

Excel のような表形式で、図の Box・Line・SDSG をリスト表示・編集できます。
画面が狭いときは **F2** で表示・非表示を切り替えられます。

### ③ キャンバス (中央) — 一番大切な作図エリア

ここが図を描く場所です。
- **マウスホイール**: 縦スクロール (Shift 押しながらで横スクロール)
- **Ctrl + ホイール**: ズーム
- **空白部分をドラッグ**: 全体をパン (移動)
- **Box・矢印を直接ドラッグ**: 移動

### ④ プロパティパネル (右)

選択した要素 (Box / Line / SDSG など) の詳細設定を編集する場所です。
- 何も選択していないときは「凡例の設定」が出ます
- Box を選択すれば「Box の設定」、Line を選択すれば「Line の設定」が出ます

### ⑤ シートタブ + ステータスバー (下)

下部には次のものがあります:
- **シートタブ**: 1 つの `.tem` ファイルに複数の図 (シート) を持てます
- **ステータスバー**: 現在のズーム倍率、選択中のアイテム数など

## 最低限覚えたい操作 5 つ

### 1. 保存する

**Ctrl + S** で保存できます。最初は名前を聞かれます。

> Mac の方は Ctrl の代わりに **⌘ (command)** キーを使います。

ファイルは `.tem` という拡張子で保存されます。
`.tem` ファイルは TEMerPlus 専用の図ファイルで、テキストエディタでも開けますが、
**TEMerPlus で開くのが基本** です。

> **自動バックアップ機能**: 保存し忘れて閉じても、ブラウザ内に
> 自動で控えが取られています。次回起動時に「自動バックアップを復元しますか？」
> と聞かれたら「はい」を選んでください。

### 2. 元に戻す / やり直し

操作を間違えても **Ctrl + Z** で元に戻せます。
やり直したいときは **Ctrl + Shift + Z** または **Ctrl + Y** です。

### 3. ズーム / フィット

- **Ctrl + ホイール**: 拡大・縮小
- リボンの「View」タブの **フィット** ボタンで、図全体を画面に合わせて表示

### 4. 選択

- **クリック**: 単一選択
- **Ctrl + クリック**: 複数選択 (追加)
- **空白部分をドラッグ**: 範囲選択 (リボン Home の「選択」モード時)
- **Ctrl + A**: 全選択

### 5. コマンドパレット (魔法のショートカット)

**Ctrl + K** で「コマンドパレット」が開きます。
やりたい操作のキーワードを入力すれば、機能を直接実行できます。

> 例:
> - 「保存」と入力 → 保存コマンドが出る、Enter で実行
> - 「グリッド」と入力 → グリッド表示の切替
> - 「凡例」と入力 → 凡例の設定を開く
>
> **どこに何があるか分からないときの最強の助け** です。

## キーボードショートカット早見表

| 操作 | ショートカット |
|---|---|
| 新規作成 | Ctrl + N |
| 開く | Ctrl + O |
| 保存 | Ctrl + S |
| 名前を付けて保存 | Ctrl + Shift + S |
| 元に戻す | Ctrl + Z |
| やり直し | Ctrl + Shift + Z または Ctrl + Y |
| 全選択 | Ctrl + A |
| コピー | Ctrl + C |
| 貼り付け | Ctrl + V |
| 複製 | Ctrl + D |
| 削除 | Delete または Backspace |
| 選択解除 | Esc |
| **コマンドパレット** | **Ctrl + K** |
| 新しいシートを追加 | Ctrl + Shift + T |

> Mac の方は `Ctrl` を `⌘ (command)` に読み替えてください。
