# 07 コマンドパレット (Ctrl+K)

[← マニュアルトップに戻る](./README.md)

> このページで分かること: Ctrl+K で開く「魔法の検索窓」の使い方。

## コマンドパレットとは

「あの機能、どこにあったっけ？」となったとき、
**Ctrl + K** (Mac は **⌘ + K**) を押すと、
全機能をテキスト検索できる窓が画面上部に開きます。

```
┌─────────────────────────────────┐
│ 🔍 コマンドを検索...                  │
├─────────────────────────────────┤
│ ファイル                                │
│   新規作成                  Ctrl+N    │
│   開く...                   Ctrl+O    │
│   保存                      Ctrl+S    │
│ 編集                                    │
│   元に戻す                  Ctrl+Z    │
│   ...                                   │
└─────────────────────────────────┘
```

## キーボード操作

| 操作 | キー |
|---|---|
| 開く | **Ctrl + K** (Mac: ⌘+K) |
| 閉じる | **Esc** |
| 選択を 1 つ下へ | **↓** |
| 選択を 1 つ上へ | **↑** |
| 実行 | **Enter** |
| マウスホバー | クリックで実行も可 |

## 検索のコツ

- **大文字小文字** は区別されません (`box` でも `Box` でも OK)
- **部分一致** で動きます (`保` でも「保存」がヒット)
- **キーワードフィールド** にも対応 (`save` でも「保存」がヒット)
- **カテゴリ名** でも検索可 (`挿入` で挿入系コマンドが絞られる)

## カテゴリと主要コマンド

50 以上のコマンドが 8 カテゴリに分かれています。

### File (ファイル)
- 新規作成 / 開く / 保存 / 名前を付けて保存

### Edit (編集)
- 元に戻す / やり直し
- 全選択 / コピー / 貼り付け / 複製 / 削除 / 選択解除
- Box を文字に合わせる (幅・高さ / 幅のみ / 高さのみ)

### Insert (挿入)
- Box: 8 種別それぞれの追加
- SD / SG (選択 Box / Line に紐付け)
- Line (RLine / XLine、選択 Box を順につなぐ)
- 2 選択 Box の間に挿入
- 時期ラベル...

### View (表示)
- グリッド / スナップ / 用紙枠 ON-OFF
- 全 ID バッジ切替 / Box・SDSG・Line ID 個別
- 上下ルーラー / 凡例 表示切替
- フィット (全要素 / 幅 / 高さ)

### Align (整列・サイズ)
- 整列 7 種 (左 / 中央 / 右 / 上 / 中央 / 下 / 水平等間隔)
- サイズ揃え 4 種 (幅 / 高さ / 両方 / 文字サイズ)

### Export (出力)
- 出力プレビュー...
- 論文レポート...

### Tools (ツール)
- 設定を開く...
- 設定: 全体 / スナップ / Box スタイル / タイプラベル / 非可逆的時間 / 時期区分 / 凡例 / SD/SG 配置 / プロジェクト (それぞれ直接ジャンプ)
- CSV インポート...
- コンテンツをシフト...
- リサイズ...

### Help (ヘルプ)
- バージョン情報

## ON/OFF が今どっちか分からない時

View タブのトグル系コマンドは、ラベルに **現在の状態を反映** します。

| 表示例 | 意味 |
|---|---|
| グリッド: ON | 現在オン → 押せばオフになる |
| グリッド: OFF | 現在オフ → 押せばオンになる |
| ID: ✓ (リボン側) | 全 ID 表示中 |
| ID: … | 部分的に表示中 |
| ID: (空) | すべて非表示 |

## 設定タブへの直接ジャンプ

設定の「凡例」を開きたいとき、以下のように 2 段階のクリックが普通必要:
1. File → 設定
2. 設定ダイアログで「凡例」タブをクリック

コマンドパレットなら 1 ステップ:
- **Ctrl + K** → `凡例` と入力 → Enter (「設定: 凡例」を選んで)

「**設定: タイプラベル**」「**設定: SD/SG 配置**」など、9 タブ全てに直接ジャンプ可能です。

## こんなときに便利

### 機能の名前は知ってるけどボタンがどこか分からない
→ Ctrl+K + キーワード で一発。

### マウス操作が苦手 / 手を動かしたくない
→ ほぼ全機能をキーボードだけで実行可。

### 同じ操作を繰り返したい
→ Ctrl+K → 履歴は残らないので毎回入力するが、短いキーワードで実行できる。

### 「あの機能あったよな…」を思い出すきっかけに
→ 「カラ」と打って何が出るか確認、のような使い方もできます。

## ショートカットの併用

コマンドパレットを開かずに直接実行できる定番ショートカット:

| 機能 | ショートカット |
|---|---|
| 保存 | Ctrl+S |
| 元に戻す | Ctrl+Z |
| 全選択 | Ctrl+A |
| 削除 | Delete |
| 複製 | Ctrl+D |

これらは **Ctrl+K を経由せず即実行** できる主要機能です。
それ以外はパレット経由が便利。
