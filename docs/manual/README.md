# TEMerPlus マニュアル & チュートリアル

TEMerPlus は **複線径路等至性モデル (TEM)** の図を描くためのツールです。
このマニュアルは、PC の操作にあまり慣れていない方や、TEM を初めて学ぶ方にも分かるよう、
ひとつひとつ丁寧に説明します。

## このマニュアルの読み方

「まず何から読めばいいの？」という方は、次の順番がおすすめです。

1. **[00 はじめに — TEM って何？](./00_what-is-tem.md)**
   TEM がどんな図で、何のために描くのかを、たとえ話を使って説明します。
2. **[01 TEMerPlus を起動する](./01_getting-started.md)**
   ブラウザでアプリを開く、画面を眺める、何ができるかを把握します。
3. **[02 はじめての図を描いてみよう (チュートリアル)](./02_tutorial-first-diagram.md)**
   実際に「進路選択」をテーマにした小さな TEM 図を、手順を追って描きます。
4. **[03 図の部品 (Box・Line・SD/SG) について詳しく](./03_basic-elements.md)**
   それぞれの部品の役割と使い分けを学びます。
5. **[04 装飾とスタイル](./04_styling.md)**
   色・フォント・サイズなどを調整して、見やすい図に仕上げます。
6. **[05 出力 (画像・PDF・PowerPoint)](./05_export.md)**
   完成した図を発表資料や論文に貼り付ける方法。
7. **[06 設定ダイアログを使いこなす](./06_settings-reference.md)**
   詳細な見栄えや既定値を調整したいときに参照する辞典。
8. **[07 コマンドパレット (Ctrl+K)](./07_command-palette.md)**
   どこに機能があるか分からなくなったらこれを開けば見つかります。
9. **[08 困ったら / よくある質問](./08_troubleshooting.md)**

## 用語のミニ辞典

聞きなれない言葉が出てきたら、ここで意味を確認してください。

| 用語 | 読み方 | 意味 |
|---|---|---|
| **TEM** | テム | 複線径路等至性モデル。人がたどる人生・経験の道筋を、複数の枝分かれする径路として描く図 |
| **径路** | けいろ | 「経過したみち」のこと。図の中で人がたどる時間的な流れを線で表す |
| **等至点 (EFP)** | とうしてん | 多くの人が結局そこへ収束する地点 (例: 「進学を決定」) |
| **両極化分岐点 (BFP)** | りょうきょくか…ぶんきてん | 径路がはっきり 2 方向以上に分かれる重要なポイント |
| **必須通過点 (OPP)** | ひっすつうかてん | 誰もがそこを通らないと先に進めない地点 (例: 「受験を申し込む」) |
| **社会的方向 (SD)** | エス・ディー | 人を「あるべき道」に押す社会的な圧力 (例: 「親の期待」) |
| **社会的助成 (SG)** | エス・ジー | 人を「あるべき道」に進めるよう支える社会的サポート (例: 「先生の励まし」) |
| **Box** | ボックス | 図の中の四角や楕円の形のこと。TEM では各種の地点 (EFP, BFP 等) を表す |
| **Line** | ライン | 矢印のこと。Box と Box をつないで径路を表す |
| **径路 (Line) の種類** | | 実線の RLine = 実際にたどった径路、点線の XLine = ありえたが選ばれなかった径路 |

> **TEM 初心者の方へ**: 全ての用語をいきなり覚える必要はありません。
> [00 はじめに](./00_what-is-tem.md) で「等至点」「BFP」など主要な概念を、
> 具体例を使って説明します。
