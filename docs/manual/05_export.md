# 05 出力 (画像・PDF・PowerPoint)

[← マニュアルトップに戻る](./README.md)

> このページで分かること: 完成した図を、発表資料や論文に使える形式で書き出す方法。

## 出力できる形式

| 形式 | 拡張子 | 用途 | 特徴 |
|---|---|---|---|
| **PNG** | `.png` | Word / メール / Web | 普通の画像。最も汎用 |
| **SVG** | `.svg` | Illustrator / Inkscape / Web | 拡大しても綺麗。各図形が編集可能 |
| **PDF** | `.pdf` | 印刷 / 配布 | 用紙サイズ通りに出る |
| **PPTX** | `.pptx` | PowerPoint | 各図形を PowerPoint 上で再編集可 |
| **印刷** | (プリンター直送) | 紙に印刷 | ブラウザの印刷ダイアログを使う |

## 出力プレビューを開く

3 つの方法があります。

| 方法 | 操作 |
|---|---|
| リボンから | **File タブ → 出力** ボタン |
| キーボード | **Ctrl + K** → 「出力」と入力 → Enter |
| 直接 | (ショートカットは設定によって変わります) |

出力プレビューダイアログが開きます。

## ダイアログの使い方

ダイアログは **左パネル (設定)** と **右パネル (プレビュー)** に分かれます。

### 左パネルの構造 (折りたたみセクション)

| セクション | 何を設定するか |
|---|---|
| **出力方式** (上部) | PNG / SVG / PDF / PPTX / 印刷 から選ぶ |
| **印刷モードタブ** | 単一印刷 / 複数印刷 (分割) |
| **用紙とフィット** | 用紙サイズ・フィットモード・余白 |
| **ページ分割** (複数モード時) | N 分割 / 続マーカー / オーバーラップ |
| **微調整** | 位置・全体倍率・文字サイズ ± / Box 間距離 |
| **出力設定** | 背景色・グリッド・用紙枠・ID バッジ含めるか |

各セクションは折りたためるので、頻繁に使う設定だけ展開しておけます。

### 右パネルのプレビュー

リアルタイムで「実際の出力結果」を表示します。
- ドラッグで位置調整 (= 微調整の位置 X/Y に反映)
- ホイールでズーム (= 全体倍率に反映)

> **プレビューと実出力は完全に同じ** ではありません。文字レンダリング等が
> 微妙に異なる場合があります。最終確認は実出力ファイルで。

## 形式別の使い分け

### PNG にしたい
- Word に貼る、メールに添付するなら **PNG**。
- 解像度は内部で 2 倍 (いわゆる Retina) で書き出されるので、
  表示がぼやけることはほぼありません。

### SVG にしたい
- 「Illustrator で後から編集したい」「拡大しても劣化しないようにしたい」なら **SVG**。
- TEMerPlus は **完全ベクター SVG** で出力します。
  各 Box・矢印・五角形・テキストが個別の図形オブジェクトとして編集可能です。

### PDF にしたい
- 印刷 / 配布する PDF が必要なら。用紙サイズ通り、余白指定も可。

### PPTX にしたい
- PowerPoint に取り込んで「あとちょっと位置を動かしたい」「スライドの一部にしたい」なら **PPTX**。
- 各 Box・矢印が PowerPoint 上で個別の図形として編集できます。
- 二重線の Box (EFP 等) も PowerPoint ネイティブの二重線で出ます。

### 印刷したい
- 「**🖨️ プリンターで印刷**」を選ぶと、新規ウィンドウが開いてブラウザの印刷ダイアログが出ます。
- 用紙サイズ・向き・カラー / モノクロ等はブラウザの印刷ダイアログ側で設定。

## 用紙サイズとフィット

### 用紙サイズ
A4 縦 / 横、A3 縦 / 横、16:9、4:3 などが選べます。
描画する図のレイアウトに応じて自動で初期値が決まります (横型 → A4 横、等)。

### フィットモード

| モード | 動作 |
|---|---|
| **両方に収める** (既定) | 用紙の縦横に合わせて図を縮小 |
| **横だけ収める** | 横幅を合わせる、縦は超えてもよい |
| **縦だけ収める** | 縦を合わせる、横は超えてもよい |
| **短辺だけ収める** | 複数印刷のとき、短辺を 1 ページに収める |
| **手動 (倍率指定)** | 任意の倍率を指定 |

### 用紙内余白比率
用紙の端からの余白の比率 (0 〜 0.3)。0.1 (= 10%) が既定。

## 複数ページに分割して印刷 (大きな図用)

横長の図が A4 1 枚に収まらない場合、**複数印刷モード** で N ページに分割できます。

1. 上部の「**複数印刷 (分割)**」タブを選択。
2. **ページ分割** セクションで:
   - **分割数**: 2 〜 20 ページ
   - **分割方式**:
     - **オーバーラップ** (既定): 単純にクロップ。各ページ境界がそのまま現れる
     - **SPEC方式 (PPTX 限定)**: ページ境界の Box を複製し、続マーカー (→続 / 続→) を入れる
   - **オーバーラップ量**: ページ間の重なり (px)
3. 下部の「📋 プレビュー: 1/3」のような表示でページを切り替え確認可。

> 複数ページ出力は **PNG / SVG 形式の場合 ZIP** にまとめてダウンロードされます
> (各ページが個別ファイル)。**PDF・PPTX** は 1 ファイル内の複数ページ。

## 微調整セクションの細かいパラメータ

「ちょっと位置をずらしたい」「全体的に文字を大きく」「Box 間隔を広げたい」など。

| パラメータ | 効果 |
|---|---|
| 位置 X / Y (px) | 全要素を平行移動 |
| 全体倍率 (手動フィット時) | 全体スケール |
| 文字サイズ ± / × | 文字の大きさを一括加算 / 倍率変更 |
| 枠線太さ ± / 線太さ ± | 線の太さを一括調整 |
| Box 間距離 横 × / 縦 × | Box 間の間隔を伸縮 (Box サイズは保持) |
| タイプラベル / サブラベル / 時間矢印 / 凡例 / 時期区分 文字 ± | 各種ラベルの文字サイズ調整 |
| 時期区分 線太さ ± | 時期区分線の太さ |
| 用紙中央に自動配置 | 図を用紙中央に寄せる (既定 ON) |

## 出力設定セクション

| 設定 | 効果 |
|---|---|
| 背景 | 白 / 透明 (PNG / SVG のみ有効) |
| グリッドを含める | 背景の方眼を含めるか (既定 OFF) |
| 用紙枠を含める | 用紙ガイドの線を含めるか (既定 OFF) |
| **ID バッジを含める** | Box / SDSG / Line の ID 表示 (個別 ON/OFF 可) |
| PDF マージン (PDF 時のみ) | 用紙余白 (inch) |

> **ID バッジ** は普段は OFF。「査読用に箇所を特定しやすくしたい」とか
> 「研究者間でやり取りするときに参照しやすくしたい」場合に ON にすると、
> 各 Box / SDSG / Line に短い ID が振られて出力されます。

## 進捗バーとキャンセル

複数ページの出力など時間がかかる場合、ダイアログ中央に **進捗バー** が出ます。

```
出力中...
ページ 3 / 7 をクロップ中
▓▓▓▓▓░░░░░░░ 42%
[ キャンセル ]
```

途中でやめたいときは **キャンセル** ボタンで停止できます。

## 出力後の確認

出力ファイルは:
- ブラウザのダウンロードフォルダ (デフォルト)
- File System Access API 対応ブラウザ (Chrome / Edge) なら、ダイアログで保存先を指定可

ダウンロードした PNG / SVG / PDF / PPTX を開いて、実際の見た目を確認してください。

## よくある質問

### Q. 文字が小さくて読めない
A. 微調整セクションの「文字サイズ +」を `2` や `4` にしてみてください。
   または用紙サイズを大きいものに変更。

### Q. 余白が大きすぎる / 小さすぎる
A. 「用紙内余白比率」を調整 (0.05 など小さくすると詰まる)。
   PDF の場合は「PDF マージン」で調整。

### Q. PPTX で開いたら矢印が線分にバラけている
A. L字 / 曲線の矢印は内部的に複数の線分に分割されます。
   PowerPoint のグループ化機能でまとめると扱いやすくなります。

### Q. 出力がエラーになる
A. ブラウザの開発者コンソール (F12 → Console タブ) にエラー詳細が出ています。
   多くは要素が多すぎて重い or ブラウザのメモリ不足です。
   Box を減らす / 解像度を下げる等で対処できます。
