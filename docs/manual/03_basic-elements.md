# 03 図の部品について詳しく (Box・Line・SD/SG)

[← マニュアルトップに戻る](./README.md)

> このページで分かること: TEM 図に登場する全ての部品の意味と使い方。

## 部品一覧

TEMerPlus で扱える部品は大きく **3 系統** に分かれます:

| 系統 | 何を表すか | 形 |
|---|---|---|
| **Box** | 地点・状態 (径路上のポイント) | 四角・楕円 (8 種類) |
| **Line** | 径路 (時間的な流れ) | 矢印 (実線・点線) |
| **SD/SG** | 社会的圧力・支援 | 五角形 (上向き △ / 下向き ▽) |

## Box の 8 種類

「地点」を意味する Box には、TEM の概念に対応した 8 種類があります。
それぞれ **見た目** で区別できるよう、形・線種・線太さが異なります。

| 種類 | 略称 | 形 | 意味 | 使う場面 |
|---|---|---|---|---|
| **通常** | normal | 普通の四角 | 一般的な地点・状態 | 特別な意味がないステップ全般 |
| **必須通過点** | OPP | 太線の四角 | 全員が通る関門 | 「受験する」「免許を取る」など全員が通るイベント |
| **両極化分岐点** | BFP | 角丸の四角 | 径路がはっきり分かれる | 「進学 or 就職を選ぶ」のような重要分岐 |
| **等至点** | EFP | 二重線の四角 | 多くの人が至る収束地点 | 「大学合格」のようなゴール |
| **両極化等至点** | P-EFP | 二重線+点線の四角 | EFP の対極 | 「不合格 (浪人)」のような EFP の裏側 |
| **第二等至点** | 2nd EFP | 二重線の四角 | EFP の次の段階 | 「卒業」「就職」など |
| **第二両極化等至点** | P-2nd EFP | 二重線+点線の四角 | P-EFP の次の段階 | EFP の派生概念 |
| **潜在経験** | annotation | 点線の四角 | 想像上・潜在的な経験 | 「もし〜だったら」の心の中の経験 |

> **覚えるコツ**:
> - 太線 = 強制感 (OPP)
> - 角丸 = 分岐感 (BFP)
> - 二重線 = ゴール感 (EFP)
> - 点線 = 仮定 / 潜在 (P-EFP, annotation)

### 種類を後から変える

Box を選択 → プロパティパネルの **基本タブ** → 「種別」のドロップダウンで変更可。

### タイプラベル (連番) について

OPP・BFP・EFP などの Box には自動的に **「OPP-1」「BFP-2」「EFP」「2nd EFP」**
のような **タイプラベル** が表示されます。

- 同じ種別が複数あると、自動で連番が付きます (`OPP-1`, `OPP-2`, …)
- EFP / 2nd EFP は「EFP」「2nd EFP」「3rd EFP」のように序数表記
- ラベルを表示したくない場合は、Box プロパティの **タイプラベル** タブで個別に OFF
- 全種別をまとめて切り替えたい場合は、リボン **View → タイプラベル ✓** ボタン

## Line (矢印) の 2 種類

| 種類 | 略称 | 線 | 意味 |
|---|---|---|---|
| **実線矢印** | RLine | 実線 | 実際にたどった径路 |
| **点線矢印** | XLine | 点線 | ありえたが選ばれなかった径路 |

### Line の追加方法

#### 方法 1: ドラッグ
1. Box の右端に近づくと青い丸が出る。
2. それを掴んで別の Box にドラッグ → 矢印が引かれる。

#### 方法 2: 一括追加
1. 始点 Box → 終点 Box の順に **Ctrl+クリック** で複数選択。
2. **Insert タブ → Line / 矢印** から「RLine 追加」または「XLine 追加」。
3. 選択順に従って隣同士に矢印が引かれる。

### Line の形状 (3 種類)

矢印の **道筋** には 3 種類あります:

| 形状 | 名前 | 用途 |
|---|---|---|
| **straight** | 直線 | 既定。最短距離で結ぶ |
| **elbow** | L字 | 90° で折れ曲がる。縦の差を階段状に表現 |
| **curve** | 曲線 | なめらかに曲がる。並列の矢印が見やすくなる |

形状は **Line を選択 → プロパティパネル → 基本タブ → 形状** で切り替えられます。

> **L字を使うと**: 折れる位置を後でドラッグ調整できます (オレンジの丸ハンドル)。
> **曲線を使うと**: 制御点が現れて、自由に曲がり方を調整できます (青い丸ハンドル)。

### 角度モード (高度)

「BFP からの矢印を全部 +30° で引きたい」のようなケースは、
**角度モード** をオンにすると一定角度で描画できます。

- プロパティパネル → 端点タブ → 「角度モード」 ✓
- 角度: -85° ~ +85° で指定

## SD / SG (社会的方向 / 社会的助成)

人の選択に影響を与える社会的な力を表す五角形です。

| 記号 | 名前 | 何を表すか | 例 |
|---|---|---|---|
| **SD** | 社会的方向 | 「あるべき道」に押し出す圧力 | 「親が進学を強く望む」 |
| **SG** | 社会的助成 | 「あるべき道」を進めるよう支える | 「先生が応援する」 |

### 配置の慣例

- **横型レイアウト** (時間軸 = 横): SD は **下** / SG は **上** に配置
- **縦型レイアウト** (時間軸 = 縦): SD は **右** / SG は **左** に配置

矢印の向きで「圧力」「助成」の方向感を表します。

### 追加方法

1. SD/SG を関連付けたい Box (または Line) を **クリックして選択**。
2. **Insert タブ → SD/SG 作成** から「SD 追加」または「SG 追加」。
3. 選択した Box の上 (SG) または下 (SD) に五角形が現れる。

### 2 つの Box の間に配置 (between モード)

「OPP-1 と OPP-2 の間に SG を配置したい」というケースは:

1. 2 つの Box を **Ctrl+クリック** で順に選択。
2. **Insert タブ → SD/SG 作成** から「**SD (2 アイテム間)**」または「**SG (2 アイテム間)**」。
3. 2 つの Box の間にスパンする五角形が配置される。

### 帯モード (band) — 同じ位置に複数 SDSG があるとき

複数の SDSG が同じ時間位置に並ぶと、見た目がぐちゃぐちゃになりがちです。
そんなとき **帯モード** に切り替えると、専用の「帯」内に整然と整列されます。

#### 帯モードへの切替

1. 帯にしたい SDSG を選択。
2. **Insert タブ → SD/SG 配置** から「**帯モードへ**」をクリック。
3. SD は上部 (or 右側) の帯、SG は下部 (or 左側) の帯に自動配置。

#### 帯の細かい調整

詳しくは [06 設定ダイアログ](./06_settings-reference.md#sdsg-配置) の **SD/SG 配置** タブで:
- 帯の位置・高さ
- row (列) の自動整列
- 帯外にはみ出した SDSG の警告 (赤枠)

を設定できます。

## サブラベルとタイプラベル

Box / SDSG には、本体ラベル以外に 2 種類の追加表示があります。

### タイプラベル
- 種別を表す自動表示 (例: `OPP-1`, `EFP`, `SD1`)
- Box 上部 / 左に表示
- プロパティパネル → タイプラベル タブで色・サイズを調整可能

### サブラベル
- 任意の補足情報 (例: 協力者ID `P03`、年齢 `21歳` など)
- Box 下部 / 右に表示
- プロパティパネル → サブラベル タブで設定

```
       [タイプラベル: OPP-1]
       ┌──────────┐
       │  Box 本体ラベル  │
       └──────────┘
       [サブラベル: P03 (21歳)]
```

## 各部品の関係性まとめ

```
時間軸 →
                  ┌──── SG (上の帯) ────┐
                  │   △ 助成1   △ 助成2  │
                  └─────────────────┘

[Box1] ──RLine──→ [Box2] ──RLine──→ [Box3] ──RLine──→ [EFP]
                          ╲
                           ╲ XLine (ありえた径路)
                            ╲
                             [P-EFP]

                  ┌──── SD (下の帯) ────┐
                  │   ▽ 圧力1   ▽ 圧力2  │
                  └─────────────────┘
```
