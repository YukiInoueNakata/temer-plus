# 04 装飾とスタイル

[← マニュアルトップに戻る](./README.md)

> このページで分かること: 色・フォント・サイズなどを調整して、見やすく綺麗な図に仕上げる方法。

## 装飾は **プロパティパネル** から

部品を 1 つ選択するか複数選択すると、画面右側のプロパティパネルに
タブ形式で設定が現れます。

### Box・SDSG のタブ構成 (5 タブ)

| タブ | 主な内容 |
|---|---|
| **基本** | ID・種別・サイズ・位置・IDバッジ |
| **ラベル** | 本体テキストの装飾 (フォント / 色 / 装飾 / 揃え) |
| **タイプラベル** | 種別バッジ (例: OPP-1) の装飾 |
| **サブラベル** | 補足表示 (協力者 ID 等) の装飾 |
| **自動調整** | 文字に合わせる / 揃える / 論文レポート用説明文 |

### Line のタブ構成 (3 タブ)

| タブ | 主な内容 |
|---|---|
| **基本** | 線種・形状 (直線/L字/曲線)・線色・線太さ |
| **端点** | 角度モード・始点終点オフセット・マージン |
| **詳細** | ID 表示・並列位置・論文レポート用説明文・IDバッジ |

## ラベルタブの装飾項目を全部見る

Box や SDSG を選択して **ラベルタブ** を開くと、次の項目があります。

| 項目 | 説明 | 例 |
|---|---|---|
| **本体テキスト** | Box の中の文字 | `進学決意` |
| **フォント** | 書体 | UI 既定 / メイリオ / ヒラギノ … |
| **フォントサイズ** | 文字の大きさ (px) | 13 (既定) |
| **装飾** | B (太字) / I (斜体) / U (下線) のトグル | |
| **文字色** | テキストの色 | #222 (既定の濃いグレー) |
| **背景色** | Box 内側の塗りつぶし色 | 白 (既定) |
| **枠線色** | Box の輪郭の色 | #222 (既定) |
| **テキスト方向** | 横書き / 縦書き | 既定: 横書き |
| **半角英数の向き** | 縦書き時に英数字を立てるか倒すか | 既定: 立てる |
| **左右の揃え** | 左 / 中央 / 右 | |
| **上下の揃え** | 上 / 中央 / 下 | |

## 色の選び方

色を変更したい項目をクリックすると **カラーピッカー** が開きます。

- 上部の色マスから選ぶ (16 色のパレット)
- 下部の HEX 入力で `#FF0000` のように直接指定
- 「**透明 / なし**」が選択肢にある場合 (背景色・枠線色等)、塗らない選択も可能
- 「**既定 (#222)**」と書かれた選択肢は、設定値の既定 (デフォルト) を使う意味

> **色を統一したいときのコツ**:
> 設定ダイアログ ([06 設定](./06_settings-reference.md#box-スタイル)) の
> **「Box スタイル」** タブで、**Box 種別ごと** の既定色を一括変更できます。
> 個別の Box ではなくスタイル全体を変えるならこちらが便利です。

## 複数選択でまとめて変更

`Ctrl+クリック` や `Ctrl+A` で複数の Box を選択すると、
プロパティパネルの値が **共通値** で表示されます。

- 全部同じ値: その値が表示される
- 異なる値が混ざっている: `（混在）` と表示される

ここで値を変更すると、**選択中すべて** に一括適用されます。

> 「全部のフォントサイズを 14 にしたい」 → Ctrl+A → ラベルタブのフォントサイズを 14 に変更 → 一括反映。

## サイズを揃える

「Box の幅・高さがバラバラで揃えたい」という場面では、
リボン **Home タブ → 整列・サイズ** にあるボタンが便利です。

| ボタン | 動作 |
|---|---|
| 幅を揃える | 選択中の Box の幅を、先頭の Box に合わせる |
| 高さを揃える | 同様に高さを合わせる |
| サイズを揃える | 幅と高さの両方を合わせる |
| 文字サイズ揃え | フォントサイズを先頭に揃える |
| 文字に合わせる (自動調整タブ) | Box 自体を中の文字にぴったり合わせる |

## 整列ボタン

| ボタン | 動作 |
|---|---|
| 左揃え / 中央揃え / 右揃え | X 座標を揃える |
| 上揃え / 中央揃え / 下揃え | Y 座標を揃える |
| 水平等間隔 | 横方向の間隔を均等にする |

## グリッドとスナップ

きちんと並べたいときは、グリッドとスナップを活用します。

### グリッド表示
- リボン **View タブ → グリッド ✓** で背景に方眼が表示される
- 既定のグリッド間隔は設定ダイアログから変更可能

### スナップ
- リボン **View タブ → スナップ ✓** で、Box をドラッグすると
  グリッドや他の Box の端に **吸着** する
- 整列が自動で取りやすい

> **スナップ設定** は **設定ダイアログ → スナップ** タブで:
> - グリッド間隔
> - 距離スナップ (他の Box との間隔)
> - 整列ガイド (赤い線で揃いを表示)

## 縦書き

レイアウトを **縦型** にすると、時間軸が上→下に流れます。
Box の本体テキストも自動で縦書きになります。

### レイアウトの切替
**設定ダイアログ → 全体 → 表示 → レイアウト** で「縦型」を選択。

### 縦書き時の半角英数の向き
- **upright (立てる)**: 1 文字ずつ縦に積む。`A` `B` `C` のように。読みやすい (既定)
- **mixed (倒す)**: 90° 回転して横倒し。伝統的な縦書きスタイル

切替は **ラベルタブ → 半角英数の向き** で個別に。

## 既定値を変える (全 Box に適用)

毎回フォントを変えるのが面倒な場合は、設定で既定を変えます。

1. **Ctrl + K** → 「**設定**」と入力 → Enter
2. **プロジェクト** タブ → 「**既定値**」セクション
3. 「既定フォント」「既定サイズ」を変更

これ以降に追加する Box は、変更後の既定で作成されます。
(既存の Box には影響しません。それは個別に変更してください)

## まとめ: 装飾のおすすめ手順

1. **まず内容を全部入れる** — 装飾のことは考えずに、Box と矢印で構造を作る
2. **整列・サイズで形を整える** — 一直線に並べ、サイズを統一
3. **既定値を 1 度だけ調整** — 設定の「既定フォント」「既定サイズ」を全体好みに
4. **個別の強調だけ手で変える** — 重要な Box だけ色を付ける、文字を大きくする等

「全部を 1 つ 1 つ手で装飾」しようとすると消耗します。一括変更を活用しましょう。
