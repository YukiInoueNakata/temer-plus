# 02 はじめての図を描いてみよう (チュートリアル)

[← マニュアルトップに戻る](./README.md)

> このページで分かること: 実際に手を動かして、簡単な TEM 図を最初から最後まで作ります。

## このチュートリアルで作る図

「ある大学生 B さんの **研究テーマ決め** の経緯」を題材にします。

```
        SG (指導教員のヒント)
          ▲
[興味分野]→[文献検索]→[テーマ案]→→[ゼミ発表]→[テーマ確定]
                              ↘                ↗
                                  XLine (志望変更しなかった径路)
```

完成形のイメージ:
- 主な径路: 「興味分野 → 文献検索 → テーマ案 → ゼミ発表 → テーマ確定」
- 「テーマ案」が **BFP** (ここで径路が分かれる)
- 「テーマ確定」が **EFP** (最終的な収束点)
- 「指導教員のヒント」が **SG** で、「ゼミ発表」を支える
- 1 本だけ点線の径路 (XLine) で、「もし変更しなかったら」を表現

> このチュートリアルは **約 20 分** で完成できます。
> 途中で困ったら [08 困ったら / よくある質問](./08_troubleshooting.md) を参照してください。

## ステップ 1: 新規ファイルを作成

1. アプリを開いた直後の状態でも構いません。あるいは **Ctrl + N** で新規作成。
2. リボン上部の **File** タブをクリック → **新規** ボタンを押す。
3. 「未保存の変更があります…」と聞かれたら、最初なので「はい」で OK。

これで真っ白なキャンバスが表示されます。

## ステップ 2: 最初の Box を 1 つ追加する

「興味分野」という最初の Box を作ります。

1. リボン上部の **Insert (挿入)** タブをクリック。
2. **Box** グループの中の「**通常**」(または「Box: normal を追加」と書かれた矩形ボタン) をクリック。
3. キャンバスに四角が 1 つ表示されます。

> **追加した Box の位置**: 自動的に画面の中央に置かれます。
> 後でドラッグして動かせるので、位置はあまり気にしなくて構いません。

### Box の中の文字を変える

1. 追加された四角を **ダブルクリック** します。
2. 文字入力モードになるので、`興味分野` と入力。
3. **Enter** または別の場所をクリックすると確定。

> Enter で改行されてしまった場合は、**Shift + Enter で改行**、
> **Enter のみで確定** という挙動です。

## ステップ 3: 残りの Box も追加する

同じ要領で次の Box を作ります。

| 名前 | 種別 |
|---|---|
| 文献検索 | 通常 |
| テーマ案 | **BFP** (両極化分岐点) |
| ゼミ発表 | 通常 |
| テーマ確定 | **EFP** (等至点) |

### BFP・EFP は別のボタンから

「BFP」「EFP」のボタンも Insert タブの Box グループにあります。

- **BFP**: 角丸の四角で表示されます (両極化分岐点を表す慣例)
- **EFP**: 二重線の四角で表示されます (等至点の慣例)

つまり、

1. **BFP** ボタンを押すと「テーマ案」用の角丸 Box が追加されます。
2. **EFP** ボタンを押すと「テーマ確定」用の二重線 Box が追加されます。

それぞれダブルクリックして文字を入れてください。

## ステップ 4: Box を時系列順に並べる

5 つの Box を、左から右へ時系列順に並べます。

1. Box をマウスでつかんで **ドラッグ** すると、自由に動かせます。
2. 「興味分野」を一番左、「テーマ確定」を一番右に。
3. 上下の高さは、最初は揃えておくと矢印が綺麗に描けます。

### 揃えるのが面倒なときは

複数の Box を選択して、**Home タブ → 整列・サイズ** から「上揃え」を実行すると、
選択中の Box が一直線に並びます。

> 複数選択は **Ctrl + クリック** または **Ctrl + A** (全選択)。
>
> 「サイズを揃える」ボタンも便利。すべての Box を同じ大きさに揃えられます。

## ステップ 5: 矢印 (Line) で Box をつなぐ

ここからが TEM 図の本番です。Box 同士を矢印でつないで径路を表現します。

### 方法 A: ドラッグでつなぐ (基本)

1. 「興味分野」Box の **右端** にカーソルを近づけると、青い小さな丸が出ます。
2. その丸をクリックしながら、「文献検索」Box にドラッグ。
3. 離すと矢印 (RLine = 実線) が引かれます。

同じ要領で:
- 「文献検索」 → 「テーマ案」
- 「テーマ案」 → 「ゼミ発表」
- 「ゼミ発表」 → 「テーマ確定」

### 方法 B: 一括追加 (慣れた人向け)

1. **Ctrl + クリック** で 2 つ以上の Box を順番に複数選択 (たとえば「興味分野」→「文献検索」→「テーマ案」の順で)。
2. **Insert タブ → Line / 矢印** から **「RLine 追加」** をクリック。
3. 選択順の隣同士に自動で矢印が引かれます。

> 矢印の向きは「先に選んだ Box から、後に選んだ Box へ」です。

## ステップ 6: 点線の矢印 (XLine) を追加

「もし変更しなかったら」という想像上の径路を、点線で表します。

1. 「テーマ案」を選択 → **Ctrl + クリック** で「テーマ確定」も選択。
2. **Insert タブ → Line / 矢印** から **「XLine 追加」** をクリック。
3. 点線の矢印が「テーマ案 → テーマ確定」に直接つながります。

> XLine と RLine は見た目で区別できます。実線 = 実現した径路、点線 = ありえた径路。

## ステップ 7: SG (社会的助成) を追加

「指導教員のヒント」を SG として描きます。

1. 「ゼミ発表」Box を **クリックして選択**。
2. **Insert タブ → SD/SG 作成** から **「SG 追加」** をクリック。
3. 「ゼミ発表」の上に上向き五角形 (△ 型) が現れ、デフォルトで「SG」と書かれています。

### SG の文字を変える

1. 五角形を **ダブルクリック**。
2. `指導教員のヒント` と入力 → Enter で確定。

> SG は「ゼミ発表」を支える存在として上にあります。
> 縦型レイアウトに切り替えると左右の関係に変わります。

## ステップ 8: 見やすく整える

このままだと文字が小さいかもしれません。微調整します。

### 文字サイズを変えたい

1. 文字を変更したい Box を選択。
2. 右側の **プロパティパネル** に「ラベル」というタブが見えます。
3. **ラベル** タブの中の「フォントサイズ」を変更。

### 色を付けたい

1. 同じくプロパティパネルの **ラベル** タブで「文字色」「背景色」などを変更できます。

### Box を全部いっぺんに大きくしたい

1. **Ctrl + A** で全選択。
2. プロパティパネル → **ラベル** タブ → 「フォントサイズ」を 16 などに変更。

### 整列ボタンを使う

1. **Ctrl + A** で全選択。
2. **Home タブ → 整列・サイズ** から「**上揃え**」を実行 → 全部のY座標が揃う。
3. 続けて「**水平等間隔**」を押せば、横方向も等間隔に並ぶ。

## ステップ 9: 保存する

**Ctrl + S** で保存します。

1. 最初の保存ではファイル名を聞かれます。`研究テーマ決定_練習` などと入力。
2. 保存場所は分かりやすい場所を選んでください (デスクトップなど)。
3. ブラウザによっては、何も聞かれず勝手にダウンロードフォルダに保存される場合もあります。

> **2 回目以降の Ctrl + S** は、同じ場所に上書き保存されます。
> 別名で保存したいときは **Ctrl + Shift + S** で「名前を付けて保存」。

## ステップ 10: 出力する

完成した図を画像にして、レポートやスライドに貼り付けます。

1. **File タブ → 出力** ボタン (または **Ctrl + K** で「出力プレビュー」と検索 → 実行)。
2. 出力プレビューダイアログが開きます。
3. 左側で出力形式を選択:
   - **PNG**: 普通の画像、Word / PowerPoint に貼り付けやすい
   - **SVG**: 拡大しても綺麗、Illustrator で編集可能
   - **PDF**: 印刷用
   - **PPTX**: PowerPoint ファイル、各図形が編集可能な状態で出力
4. 右側でプレビューを確認しながら、用紙サイズや余白を調整。
5. 下部の「**💾 PNG で保存**」ボタンを押して保存。

> 詳しくは [05 出力 (画像・PDF・PowerPoint)](./05_export.md) を参照。

## 完成！

これで最初の TEM 図が完成です 🎉

最終的にあなたが手にしているのは:
- `.tem` ファイル (TEMerPlus で再編集できる)
- 画像ファイル (発表や論文に使える)
